\begin{statementbox}
对于任意实数 $a, b, c \in \mathbb{R}$，有以下核心不等式结论：
\begin{enumerate}
    \item \textbf{基本形式}：$a^2 + b^2 + c^2 \geqslant ab + bc + ca$ ；
    \item \textbf{等效形式}：
    \begin{itemize}
        \item $3(a^2 + b^2 + c^2) \geqslant (a+b+c)^2$
        \item $(a+b+c)^2 \geqslant 3(ab + bc + ca)$
    \end{itemize}
    \item \textbf{等号成立条件}：当且仅当 $a = b = c$ 时，等号成立。
\end{enumerate}
\end{statementbox}
