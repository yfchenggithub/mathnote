\begin{proofbox}
\textbf{配方法（最经典证明）：}
\begin{enumerate}
    \item 将不等式各项移至左侧：$a^2 + b^2 + c^2 - ab - bc - ca \geqslant 0$ ；
    \item 全体乘以 2：$2a^2 + 2b^2 + 2c^2 - 2ab - 2bc - 2ca \geqslant 0$ ；
    \item 重新组合项：$(a^2 - 2ab + b^2) + (b^2 - 2bc + c^2) + (c^2 - 2ca + a^2) \geqslant 0$ ；
    \item 利用完全平方公式：$(a-b)^2 + (b-c)^2 + (c-a)^2 \geqslant 0$ ；
    \item 因为平方数恒大于等于 0，且只有当 $a-b=0, b-c=0, c-a=0$ 即 $a=b=c$ 时取等号，结论得证。
\end{enumerate}
\end{proofbox}
