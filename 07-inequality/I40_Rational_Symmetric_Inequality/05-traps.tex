\begin{itemize}
    \item \textbf{正数限制}：前提必须是 $a, b > 0$。如果存在负数，平方项虽为正，但分母的符号会改变不等号方向。
    \item \textbf{等号条件}：虽然本题等号在 $a=b$ 处取得，但在某些约束条件下（如 $a+2b=1$），$a$ 与 $b$ 可能无法相等，此时需改用配凑法或拉格朗日乘数法思想。
    \item \textbf{分子分母混淆}：注意是分子带平方。如果是 $\frac{a}{b^2} + \frac{b}{a^2}$，结论则完全不同。
\end{itemize}
\end{trapbox}
