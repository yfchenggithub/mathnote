\begin{trapbox}
\begin{itemize}
    \item \textbf{符号混淆}：虽然 $\ln x$ 是上凸的（方向为 $\leqslant$），但 $x\ln x$ 是下凸的（方向为 $\geqslant$）。千万不要因为看到 $\ln$ 就盲目判定方向。
    \item \textbf{定义域陷阱}：虽然 $\lim_{x\to 0^+} x\ln x = 0$，但在代数推导中必须保证 $x, y$ 严格大于 0。
    \item \textbf{结构识别}：注意识别 $x\ln x + y\ln y$ 这种"熵"结构，它经常隐藏在复杂的变元置换后。
\end{itemize}
\end{trapbox}
