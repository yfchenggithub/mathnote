\begin{itemize}
    \item \textbf{放缩过宽}：在一些极难的压轴题中，$\ln x \leqslant x-1$ 可能放缩得太厉害（即误差较大），导致无法得出目标结论。此时可能需要用到更精细的结论，如 $\ln x \leqslant \frac{x-1}{\sqrt{x}}$ 或 $\ln x \leqslant x - 1 - \frac{1}{2}(x-1)^2$。
    \item \textbf{定义域限制}：始终记住对数函数的定义域 $x > 0$。
    \item \textbf{端点取等}：在构造不等式链证明时，务必单独说明 $x=1$ 是否符合题目要求。
\end{itemize}
\end{trapbox}
