\begin{itemize}
    \item \textbf{忽略正数条件}：基本不等式 $\frac{a+b}{2} \geqslant \sqrt{ab}$ 要求 $a, b > 0$。若含负数，需考虑加绝对值或取相反数转化。
    \item \textbf{忽略"定值"条件}：在使用不等式求极值前，必须确保和或积中有一个是常数。
    \item \textbf{等号不可达}：这是最隐蔽的陷阱。必须验证 $a=b$ 时，变量是否在定义域内。
    \item \textbf{多次使用不等式}：在同一个题目中多次使用基本不等式时，必须保证每次等号成立的条件能够同时满足。
\end{itemize}
\end{trapbox}
