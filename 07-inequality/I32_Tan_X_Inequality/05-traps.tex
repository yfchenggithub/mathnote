\begin{trapbox}
\begin{itemize}
    \item \textbf{定义域爆破}：必须强调 $x < \frac{\pi}{2}$。当 $x$ 趋近于 $\frac{\pi}{2}$ 时，$\tan x$ 趋向正无穷，而 $x$ 仅为定值。
    \item \textbf{符号反转}：在 $(-\frac{\pi}{2}, 0)$ 区间内，$\tan x$ 是负数且其绝对值大于 $|x|$ 的绝对值，故此时 $\tan x < x$。
    \item \textbf{精度考量}：在处理极小的 $x$ 时，$\tan x - x$ 约等于 $\frac{x^3}{3}$，如果放缩后无法解题，可能需要三阶放缩。
\end{itemize}
\end{trapbox}
