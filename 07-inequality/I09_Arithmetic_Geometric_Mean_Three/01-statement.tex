\begin{statementbox}
对于任意非负实数 $a, b, c \in [0, +\infty)$，有以下核心不等式结论：
\begin{enumerate}
    \item \textbf{算术-几何均值不等式}：$\frac{a+b+c}{3} \geqslant \sqrt[3]{abc}$ ；
    \item \textbf{常用变形}：
    \begin{itemize}
        \item $a+b+c \geqslant 3\sqrt[3]{abc}$ （和定积最大）
        \item $abc \leqslant \left(\frac{a+b+c}{3}\right)^3$ （积定和最小）
    \end{itemize}
    \item \textbf{等号成立条件}：当且仅当 $a = b = c$ 时，等号成立。
\end{enumerate}
\end{statementbox}
