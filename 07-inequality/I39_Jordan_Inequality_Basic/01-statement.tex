\begin{statementbox}
对于任意实数 $x \in [0, \frac{\pi}{2}]$，正弦函数的割线放缩（Jordan 不等式基础型）如下：
\begin{enumerate}
    \item \textbf{核心形式}：$\sin x \geqslant \frac{2}{\pi}x$
    \item \textbf{对称形式}：当 $x \in [-\frac{\pi}{2}, 0]$ 时，$\sin x \leqslant \frac{2}{\pi}x$
    \item \textbf{等号条件}：当且仅当 $x = 0$ 或 $x = \frac{\pi}{2}$ 时，等号成立
    \item \textbf{链式关系}：在 $x \in (0, \frac{\pi}{2})$ 上，恒有 $\frac{2}{\pi}x < \sin x < x$
\end{enumerate}
\end{statementbox}
