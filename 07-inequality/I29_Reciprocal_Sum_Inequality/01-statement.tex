\begin{statementbox}
对于任意实数 $a, b$，当它们同号（即 $ab > 0$）时，存在以下核心结论：
\begin{enumerate}
    \item \textbf{核心形式}：$\frac{a}{b} + \frac{b}{a} \geqslant 2$
    \item \textbf{倒数形式}：若令 $x = \frac{a}{b}$，则结论表现为 $x + \frac{1}{x} \geqslant 2$（其中 $x > 0$）
    \item \textbf{负数情况}：若 $ab < 0$，则 $\frac{a}{b} + \frac{b}{a} \leqslant -2$
    \item \textbf{等号成立条件}：当且仅当 $a = b$ 时，等号成立。
\end{enumerate}
\end{statementbox}
