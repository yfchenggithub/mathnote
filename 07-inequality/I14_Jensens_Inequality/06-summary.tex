\begin{summarybox}
1. \textbf{判定依据}：看二阶导数 $f''(x)$ 的符号。
2. \textbf{口诀}：
    \begin{itemize}
        \item $f'' > 0$（开口向上）：函数值均值 $\geqslant$ 均值函数值。
        \item $f'' < 0$（开口向下）：函数值均值 $\leqslant$ 均值函数值。
    \end{itemize}
3. \textbf{应用实战}：处理含有 $\sum f(x_i)$ 结构的压轴证明题，这是将"分散"变量转化为"整体"均值的神技。
\end{summarybox}
