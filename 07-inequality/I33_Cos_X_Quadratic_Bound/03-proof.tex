\begin{proofbox}
\textbf{方法一：导数构造法（高考标准路径）}
\begin{enumerate}
    \item 构造辅助函数 $f(x) = \cos x + \frac{x^2}{2} - 1$ ；
    \item 一阶求导：$f'(x) = -\sin x + x$ ；
    \item 利用已知结论（结论 I31）：当 $x > 0$ 时，$\sin x < x$，故 $f'(x) > 0$；当 $x < 0$ 时，$f'(x) < 0$ ；
    \item 因此，$f(x)$ 在 $(-\infty, 0)$ 单调递减，在 $(0, +\infty)$ 单调递增 ；
    \item 函数在 $x=0$ 处取得极小值（亦为最小值）$f(0) = \cos 0 + 0 - 1 = 0$ ；
    \item 故 $f(x) \geqslant 0$ 恒成立，即 $\cos x \geqslant 1 - \frac{x^2}{2}$。
\end{enumerate}
\end{proofbox}
