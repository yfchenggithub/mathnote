\begin{itemize}
    \item \textbf{取等条件记反}：注意 $|a+b| = |a| + |b|$ 要求 $ab \geqslant 0$（同号），而 $|a-b| = |a| + |b|$ 则要求 $ab \leqslant 0$（异号）。
    \item \textbf{忽视绝对值符号}：左侧公式是 $||a|-|b||$，最外层还有一个绝对值，确保结果非负。
    \item \textbf{适用范围}：该结论在复数范围内依然成立（复数模长不等式），但证明方法需转为向量法。
\end{itemize}
\end{trapbox}
