\begin{proofbox}
\textbf{平方放缩法证明：}
\begin{enumerate}
    \item 证明 $|a+b| \leqslant |a| + |b|$ ：
    \item 两边均为非负数，平方得：$|a+b|^2 = (a+b)^2 = a^2 + 2ab + b^2$ ；
    \item 而 $(|a| + |b|)^2 = |a|^2 + 2|a||b| + |b|^2 = a^2 + 2|ab| + b^2$ ；
    \item 因为 $ab \leqslant |ab|$ 恒成立，所以 $a^2 + 2ab + b^2 \leqslant a^2 + 2|ab| + b^2$ ；
    \item 故 $|a+b|^2 \leqslant (|a| + |b|)^2$，开方得 $|a+b| \leqslant |a| + |b|$。
    \item 左侧不等式 $||a| - |b|| \leqslant |a+b|$ 可由右侧通过替换法（令 $a = (a+b) - b$）移项证得。
\end{enumerate}
\end{proofbox}
