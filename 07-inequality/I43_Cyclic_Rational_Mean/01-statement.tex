\begin{statementbox}
对于任意正实数 $a, b, c > 0$，关于循环分式的基本不等式如下：
\begin{enumerate}
    \item \textbf{核心形式}：$\frac{a}{b} + \frac{b}{c} + \frac{c}{a} \geqslant 3$
    \item \textbf{推广形式}：$\sum_{i=1}^{n} \frac{a_i}{a_{i+1}} \geqslant n$（其中 $a_{n+1} = a_1$）
    \item \textbf{等号条件}：当且仅当 $a = b = c$ 时，等号成立。
    \item \textbf{结构特征}：属于非对称循环齐次式，每一项的分子是前一项的分母。
\end{enumerate}
\end{statementbox}
