\begin{trapbox}
\begin{itemize}
    \item \textbf{方向混淆}：这是学生最容易出错的地方。一定要记住 $\tan$ 的图象是"陡峭上升"的，弦在弧上，所以 $\frac{\sum \tan}{n} \geqslant \tan(\text{avg})$。
    \item \textbf{定义域断点}：注意 $90^\circ$ 是渐近线。如果涉及的数值接近 $\pi/2$，必须注意函数趋向正无穷的情况。
    \item \textbf{锐角前提}：三角形结论中，若存在钝角，正切值会出现负数，此时该不等式链失效。
\end{itemize}
\end{trapbox}
