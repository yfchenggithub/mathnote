\begin{trapbox}
\begin{itemize}
    \item \textbf{长短轴混淆}: 在应用公式时, 务必确认 $a$ 和 $b$ 的具体数值, 不要代错平方项.
    \item \textbf{最大最小值误区}: 很多学生直观认为 $d$ 在短轴处最大, 实际上根据 $d = ab/\sqrt{r_1r_2}$, 在长轴顶点 $x_0 = a$ 时 $r_1r_2 = a^2 - c^2 = b^2$, 此时 $d = a$ 为最大值.
    \item \textbf{符号规范}: 确保所有运算符如 $\cdot$ 以及下标如 $r_1$ 均在数学环境 `$ $` 内书写, 严禁使用全角标点.
\end{itemize}
\end{trapbox}
