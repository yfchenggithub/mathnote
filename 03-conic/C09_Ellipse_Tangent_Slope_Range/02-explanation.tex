\begin{explanationbox}
该结论主要用于解决涉及 "视角" 或 "动态切线" 的离心率与最值问题:
\begin{itemize}
    \item \textbf{几何直观}: 当点 $M$ 远离椭圆时, 两条切线的夹角变小, 斜率 $k$ 的取值范围受到点 $M$ 坐标的严格约束.
    \item \textbf{方程意义}: 方程 $(a^2 - x_0^2)k^2 + 2x_0y_0k + (b^2 - y_0^2) = 0$ 是处理切线斜率乘积 ($k_1k_2$) 和斜率之和 ($k_1+k_2$) 的核心工具.
    \item \textbf{适用场景}:
    \begin{itemize}
        \item 求解过定点切线斜率的取值范围.
        \item 结合切线夹角公式求解点 $M$ 的轨迹 (如蒙日圆).
        \item 解决与斜率相关的最值或恒成立问题.
    \end{itemize}
\end{itemize}
\end{explanationbox}
