\begin{statementbox}
对于椭圆 $C: \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ ($a>b>0$) 与直线 $l: Ax + By + C = 0$。若直线与椭圆不相交，则椭圆上的点 $P(x, y)$ 到直线 $l$ 的距离 $d$ 的最值满足以下等价形式：
\begin{enumerate}
    \item \textbf{距离转化形式}：$d = \frac{|Ax + By + C|}{\sqrt{A^2 + B^2}}$；
    \item \textbf{极值公式}：$d_{max} = \frac{|C| + \sqrt{A^2a^2 + B^2b^2}}{\sqrt{A^2 + B^2}}$, $d_{min} = \frac{|C| - \sqrt{A^2a^2 + B^2b^2}}{\sqrt{A^2 + B^2}}$ (假设直线在椭圆外且 $C > 0$)；
    \item \textbf{切线常数项}：与 $l$ 平行的椭圆切线方程为 $Ax + By \pm \sqrt{A^2a^2 + B^2b^2} = 0$。
\end{enumerate}
\end{statementbox}
