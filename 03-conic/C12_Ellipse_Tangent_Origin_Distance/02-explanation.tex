\begin{explanationbox}
该性质是椭圆度量几何中的高级结论, 关联了原点、切线与焦点:
\begin{itemize}
    \item \textbf{几何直观}: 随着点 $P$ 在椭圆上移动, 虽然原点到切线的距离 $d$ 以及焦半径 $r_1, r_2$ 都在变化, 但它们的组合项 $\sqrt{r_1 r_2} \cdot d$ 始终锁定为长短轴半轴长的乘积.
    \item \textbf{结构美感}: 公式左边体现了切点的局部性质 (焦半径几何平均数) 与全局位置 (原心距) 的耦合, 右边则是椭圆的核心面积特征参数.
    \item \textbf{适用场景}:
    \begin{itemize}
        \item 求解涉及原点到切线距离的最值问题.
        \item 处理与焦半径乘积相关的综合计算题.
        \item 在解析几何压轴题中作为隐藏的等量关系辅助建模.
    \end{itemize}
\end{itemize}
\end{explanationbox}
