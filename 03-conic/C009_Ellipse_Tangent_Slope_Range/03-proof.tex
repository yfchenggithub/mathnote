\begin{proofbox}
\textbf{利用切线判定式推导}:
\begin{enumerate}
    \item 设过点 $M(x_0, y_0)$ 的切线方程为 $y - y_0 = k(x - x_0)$, 即 $y = kx + (y_0 - kx_0)$.
    \item 根据椭圆切线判定公式 (C07), 直线与椭圆相切需满足 $m^2 = a^2k^2 + b^2$.
    \item 将 $m = y_0 - kx_0$ 代入得: $(y_0 - kx_0)^2 = a^2k^2 + b^2$.
    \item 展开方程: $y_0^2 - 2x_0y_0k + x_0^2k^2 = a^2k^2 + b^2$.
    \item 移项并整理为关于 $k$ 的二次方程: $(x_0^2 - a^2)k^2 - 2x_0y_0k + (y_0^2 - b^2) = 0$.
    \item 该方程的解即为过点 $M$ 的切线斜率.
\end{enumerate}
\end{proofbox}
