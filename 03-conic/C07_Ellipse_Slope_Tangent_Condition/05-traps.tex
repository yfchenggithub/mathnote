\begin{itemize}
    \item \textbf{斜率不存在陷阱}: 公式 $m^2 = a^2k^2 + b^2$ 仅适用于斜率存在的直线. 若切线垂直于 $x$ 轴（即 $x = \pm a$）, 需单独讨论.
    \item \textbf{标准方程转化}: 在代入 $a^2, b^2$ 前, 务必先将椭圆方程化为标准形式 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$.
    \item \textbf{正负号注意}: 开方求 $m$ 时不要漏掉 $\pm$ 号, 椭圆上同一斜率的切线通常成对出现.
    \item \textbf{符号规范}: 在系统输入时, 确保使用英文半角括号 "()" 和逗号 ",", 否则无法触发 Fuse.js 的公式检索.
\end{itemize}
\end{trapbox}
