\begin{itemize}
    \item \textbf{相交性检查}：若直线与椭圆相交，距离最小值 $d_{min} = 0$，此时公式中的负号项不再适用。
    \item \textbf{系数对应关系}：公式中 $a^2$ 必须与直线方程中 $x$ 的系数 $A$ 相乘，$b^2$ 与 $y$ 的系数 $B$ 相乘。
    \item \textbf{分子绝对值}：当直线从椭圆中间穿过或距离较近时，注意 $|C| - \sqrt{A^2a^2 + B^2b^2}$ 可能为负，物理距离需取绝对值。
\end{itemize}
\end{trapbox}
