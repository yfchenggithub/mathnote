\begin{examplebox}
\textbf{例题 1：} 已知椭圆 $C: \frac{x^2}{4} + y^2 = 1$，圆 $M: (x-3)^2 + y^2 = 1$，求椭圆上的点到圆上点的最小距离。

\textit{解析：} 
1. 圆心 $M(3, 0)$，半径 $r=1$。
2. 设椭圆上点 $P(x, y)$ 到圆心距离为 $d$，则 $d^2 = (x-3)^2 + y^2$。
3. 消元得 $d^2 = (x-3)^2 + (1 - \frac{x^2}{4}) = \frac{3}{4}x^2 - 6x + 10$。
4. 由于 $x \in [-2, 2]$，函数 $g(x) = \frac{3}{4}x^2 - 6x + 10$ 在区间内单调递减。
5. 当 $x=2$ 时，$d^2_{min} = \frac{3}{4}(4) - 12 + 10 = 1$，即 $d_{min} = 1$。
6. 最小距离 $L_{min} = d_{min} - r = 1 - 1 = 0$。（说明圆与椭圆外切）。

\textbf{例题 2：} 若椭圆 $\frac{x^2}{5} + \frac{y^2}{4} = 1$ 上一点到圆 $x^2 + y^2 = 1$ 的最大距离。

\textit{解析：}
圆心在原点，椭圆长半轴 $a=\sqrt{5}$。
$d_{max} = a = \sqrt{5}$。
故 $L_{max} = d_{max} + r = \sqrt{5} + 1$。
\end{examplebox}
