\begin{proofbox}
\textbf{圆心在 $x$ 轴上的通用推导：}
\begin{enumerate}
    \item 设椭圆点 $P(x, y)$，圆心 $M(x_0, 0)$。则距离平方 $d^2 = (x-x_0)^2 + y^2$。
    \item 代入椭圆方程 $y^2 = b^2(1 - \frac{x^2}{a^2})$，得：
    $d^2 = x^2 - 2x_0x + x_0^2 + b^2 - \frac{b^2}{a^2}x^2$。
    \item 整理得二次函数 $g(x) = \frac{c^2}{a^2}x^2 - 2x_0x + x_0^2 + b^2$，其中 $x \in [-a, a]$。
    \item \textbf{结论}：通过对称轴 $x = \frac{a^2x_0}{c^2}$ 与区间 $[-a, a]$ 的位置关系，结合二次函数单调性即可确定 $d^2$ 的最值。
\end{enumerate}
\end{proofbox}
