\begin{statementbox}
对于椭圆 $C: \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ ($a>b>0$) 与圆 $O: (x-x_0)^2 + (y-y_0)^2 = r^2$。若圆与椭圆不相交，则椭圆上的点 $P(x, y)$ 到圆心 $M(x_0, y_0)$ 的距离 $d$ 与到圆上点 $Q$ 的距离 $L$ 满足以下关系：
\begin{enumerate}
    \item \textbf{距离转化}：$L_{max} = d_{max} + r$, $L_{min} = d_{min} - r$（若圆在椭圆外）或 $L_{min} = r - d_{max}$（若圆在椭圆内）；
    \item \textbf{圆心在坐标轴上的极值}：
    \begin{itemize}
        \item 若圆心为原点 $(0,0)$，则 $d_{max} = a$, $d_{min} = b$；
        \item 若圆心在 $x$ 轴上 $(x_0, 0)$，设 $f(x) = (1-\frac{b^2}{a^2})x^2 - 2x_0x + x_0^2 + b^2$，通过求其在 $[-a, a]$ 上的最值确定 $d$。
    \end{itemize}
\end{enumerate}
\end{statementbox}
