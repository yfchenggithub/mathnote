\begin{proofbox}
\textbf{利用焦半径与余弦定理推导}:
\begin{enumerate}
    \item 根据椭圆定义, $r_1 + r_2 = 2a$. 记 $|F_1F_2| = 2c$.
    \item 在 $\triangle F_1PF_2$ 中, 由余弦定理得:
    $4c^2 = r_1^2 + r_2^2 - 2r_1r_2\cos\theta = (r_1+r_2)^2 - 2r_1r_2(1+\cos\theta) = 4a^2 - 4r_1r_2\cos^2\frac{\theta}{2}$.
    \item 整理得 $r_1r_2\cos^2\frac{\theta}{2} = a^2 - c^2 = b^2$.
    \item 又由三角形面积公式 $S = \frac{1}{2}r_1r_2\sin\theta = c|y_P|$, 得 $r_1r_2 = \frac{2c|y_P|}{\sin\theta}$.
    \item 联立消去 $r_1r_2$, 可导出 $\tan\frac{\theta}{2} = \frac{c|y_P|}{b^2}$ (注意此为另一种变形形式).
    \item 结合 $r_1r_2 \le a^2$, 代入 $\cos\theta$ 公式即可证得 $\sin\frac{\theta}{2} \le e$.
\end{enumerate}
\end{proofbox}
