\begin{itemize}
    \item \textbf{计算陷阱}: 很多学生会混淆 $\sin\frac{\theta}{2}$ 与 $\cos\theta$, 在处理不等式方向时务必注意正弦函数在 $(0, \pi/2)$ 的单调性.
    \item \textbf{符号规范}: 在代码实现或文档编写时, 必须使用英文半角符号, 如 "e >= sin(theta/2)" 而非全角字符, 否则会导致解析器失效.
    \item \textbf{图形误导}: 不要误以为 $P$ 在顶点时角度最大, 实际上在短轴端点处, $P$ 到两焦点的距离最平衡, 张角才达到巅峰.
\end{itemize}
\end{trapbox}
