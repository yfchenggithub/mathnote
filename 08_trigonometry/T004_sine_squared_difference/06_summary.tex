\begin{summarybox}
\textbf{\textcolor{CSummary}{一句话核心}}  
\textbf{正弦平方差公式 $\sin^2\alpha - \sin^2\beta = \sin(\alpha-\beta)\sin(\alpha+\beta)$ 是三角恒等变换的利器，可瞬间化差为积.}

\medskip
\textbf{\textcolor{CSummary}{使用条件}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{条件 1（任意角）：} $\alpha,\beta$ 为任意实数角，无象限或范围限制.
    \item \textbf{条件 2（恒等变形）：} 等式恒成立，使用时无需额外验证前提.
\end{itemize}

\medskip
\textbf{\textcolor{CSummary}{关键提醒}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{易错点（符号与顺序）：} 差角正弦 $\sin(\alpha-\beta)$ 不可随意交换为 $\sin(\beta-\alpha)$; 与余弦平方差公式区分，后者带负号.
    \item \textbf{检查项（两角核对）：} 代入后快速验证 $\alpha-\beta$ 与 $\alpha+\beta$ 是否正确，尤其在有符号角时注意括号.
\end{itemize}
\end{summarybox}
