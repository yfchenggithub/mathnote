\begin{explanationbox}
\textbf{\textcolor{CExplanation}{一句话直觉}}  
正弦平方差公式就像三角版的平方差公式，把差变成乘积，方便化简.

\medskip
\textbf{\textcolor{CExplanation}{核心拆解}}
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.45em]
    \item[\textbf{要点一（平方差结构）：}] 左边是$\sin^2\alpha-\sin^2\beta$，类似$a^2-b^2$，但实为同名三角函数平方之差.
    \item[\textbf{要点二（积化和差形式）：}] 右边化为$\sin(\alpha-\beta)\sin(\alpha+\beta)$，差角与和角的正弦之积，可直接调用和差化积思想.
\end{description}

\medskip
\textbf{\textcolor{CExplanation}{几何本质（单位圆弦长）}}  
在单位圆中，$\sin\alpha,\sin\beta$可看作角$\alpha,\beta$终边与圆交点的纵坐标，公式联系了两角差与和的弦长关系.

\medskip
\textbf{\textcolor{CExplanation}{代数意义（恒等变形工具）}}  
该公式提供从“平方差”到“乘积”的直接转化，避免先展开$\sin^2$再积化和差的复杂过程，在三角恒等变换中衔接流畅.

\medskip
\textbf{\textcolor{CExplanation}{考点价值（化简与求值）}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{考法一（直接代换求值）：} 遇$\sin^2\alpha-\sin^2\beta$可直接代入右侧求值，减少计算量.
    \item \textbf{考法二（恒等证明）：} 结合其他公式证明复杂恒等式，突出结构转化.
\end{itemize}

\medskip
\textbf{\textcolor{CExplanation}{顿悟点}}  
\textbf{把“正弦平方差”当成“正弦差乘正弦和”的思维，能瞬间识别题目中的隐藏乘积结构.}

\medskip
\textbf{\textcolor{CExplanation}{使用场景}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{场景一（平方差出现）：} 题目中出现$\sin^2 A-\sin^2 B$，直接套用公式.
    \item \textbf{场景二（和差化积配合）：} 右侧形式暗示可用和差化积或积化和差进一步变形.
\end{itemize}
\end{explanationbox}
