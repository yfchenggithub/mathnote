\begin{explanationbox}
\textbf{深度解读：点位分布与比例记忆}
\begin{itemize}
    \item \textbf{位置顺序}：在欧拉线上，四点的排列顺序依次为：\textbf{外心(O) $\to$ 重心(G) $\to$ 九点圆心(E) $\to$ 垂心(H)}。
    \item \textbf{比例密码}：我们可以记一个简单的比例序列 $2:1:3$。
    \begin{itemize}
        \item $OG = 2 \times GE$
        \item $EH = 3 \times GE$
        \item 由此推导 $GH = 2OG$，这是竞赛中最常用的比例。
    \end{itemize}
    \item \textbf{特殊情况}：当三角形为等边三角形时，四点合一，欧拉线退化为一个点；当三角形为等腰三角形时，欧拉线即为底边上的高。
\end{itemize}
\end{explanationbox}
