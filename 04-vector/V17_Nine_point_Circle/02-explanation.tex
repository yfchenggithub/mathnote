\textbf{深度解读：九点圆的几何逻辑}
\begin{itemize}
    \item \textbf{位似变换}：九点圆本质上是以垂心 $H$ 为位似中心，将外接圆缩小为原来 $\frac{1}{2}$ 的产物。
    \item \textbf{欧拉线上的明珠}：九点圆心 $E$ 恰好位于欧拉线的中点，平衡了外心 $O$ 与垂心 $H$。
    \item \textbf{对称性}：由于 $X, Y, Z$ 是垂心与顶点的中点，它们与边中点 $D, E, F$ 构成了关于九点圆心的中心对称关系。
\end{itemize}
