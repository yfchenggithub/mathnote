\begin{examplebox}
\textbf{典型例题}：
已知 $\triangle ABC$ 的外接圆方程为 $x^2 + y^2 = 4$，垂心坐标为 $(2, 0)$。求其九点圆的圆心坐标及圆方程。

\textbf{解析}：
1. \textbf{求外心}：由方程知外心 $O(0, 0)$，半径 $R=2$。
2. \textbf{求九点圆心}：九点圆心 $E$ 是 $OH$ 的中点。
   $x_E = \frac{0+2}{2} = 1, y_E = \frac{0+0}{2} = 0$。即 $E(1, 0)$。
3. \textbf{求半径}：$R_9 = \frac{1}{2}R = \frac{1}{2} \times 2 = 1$。
4. \textbf{写方程}：$(x-1)^2 + y^2 = 1$。
\end{examplebox}
