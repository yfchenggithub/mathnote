\begin{proofbox}
利用中线向量表示及平方差公式：
\begin{enumerate}
    \item 因为 $M$ 是 $BD$ 中点，设 $\vec{AB} = \vec{a}, \vec{AD} = \vec{b}$，则：
    \[ \vec{AM} = \frac{1}{2}(\vec{a} + \vec{b}) \implies (\vec{a} + \vec{b})^2 = 4|\vec{AM}|^2 \]
    \item 向量 $\vec{DB} = \vec{a} - \vec{b}$，则：
    \[ (\vec{a} - \vec{b})^2 = |\vec{DB}|^2 \]
    \item 代入极化恒等式基本型：
    \[ \vec{a} \cdot \vec{b} = \frac{1}{4} [(\vec{a} + \vec{b})^2 - (\vec{a} - \vec{b})^2] \]
    \item 结合上述两式：
    \[ \vec{a} \cdot \vec{b} = \frac{1}{4} [4|\vec{AM}|^2 - |\vec{DB}|^2] = |\vec{AM}|^2 - \frac{1}{4}|\vec{DB}|^2 \]
\end{enumerate}
\end{proofbox}
