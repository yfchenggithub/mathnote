\begin{trapbox}
\begin{itemize}
    \item \textbf{锐钝角区别}：当 $\triangle ABC$ 为钝角三角形时，垂心 $H$ 在外部。此时 $H$ 不是垂足三角形的内心，而是其一个顶点，而钝角顶点才是垂足三角形的内心。
    \item \textbf{垂足位置}：在直角三角形中，垂心与直角顶点重合，此时垂足三角形退化为一条线段，上述性质需特殊处理。
    \item \textbf{内心混淆}：不要误认为垂心是原三角形的内心。垂心是原三角形的"高线交点"，只有在垂足三角形这个特定视角下，它才具备内心的性质。
\end{itemize}
\end{trapbox}
