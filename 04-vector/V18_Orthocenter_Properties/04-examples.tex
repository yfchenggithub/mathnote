\begin{examplebox}
\textbf{典型例题}：
在锐角 $\triangle ABC$ 中，$H$ 为垂心，若 $\triangle ABC$ 的外接圆半径为 $R$，求 $\triangle HBC$ 的外接圆半径 $R'$。

\textbf{解析}：
1. 利用垂心组性质，$A$ 是 $\triangle HBC$ 的垂心。
2. 垂心关于边的对称点 $H'$ 在外接圆上，故 $\triangle HBC$ 与 $\triangle H'BC$ 全等（关于 $BC$ 对称）。
3. $\triangle H'BC$ 的顶点均在原外接圆上，故其外心即为 $O$，半径为 $R$。
4. \textbf{结论}：$R' = R$。
\textit{注：垂心组构成的四个三角形，其外接圆半径全部相等。}
\end{examplebox}
