\begin{explanationbox}
\textbf{定理深度解读}：
\begin{itemize}
    \item \textbf{三点共线定理的推广}：当 $k=1$ 时，即为著名的"三点共线定理" $\lambda + \mu = 1$。
    \item \textbf{"等高"的物理意义}：若把 $O$ 看作原点，直线 $AB$ 看作高度为 1 的水平线，那么 $\lambda + \mu = k$ 实际上定义了一个线性函数，每一条平行于 $AB$ 的直线都是该函数的"等值线"。
    \item \textbf{比例转化}：若 $\vec{OP} = \lambda\vec{OA} + \mu\vec{OB}$，且 $P$ 落在某条等和线上，则有 $\vec{OP} = k \left( \frac{\lambda}{k}\vec{OA} + \frac{\mu}{k}\vec{OB} \right)$。括号内系数和为 1，代表在直线 $AB$ 上的投影点，而 $k$ 则是伸缩比例。
\end{itemize}
\end{explanationbox}
