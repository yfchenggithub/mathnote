\begin{itemize}
    \item \textbf{起点混淆}：注意 $\vec{GA}+\vec{GB}+\vec{GC}=\vec{0}$ 的起点必须是重心本身。如果题目给出 $\vec{MA}+\vec{MB}+\vec{MC}=3\vec{MG}$，要能迅速识别出 $G$ 是重心。
    \item \textbf{与内心混淆}：重心系数是 $1:1:1$，内心系数是边长 $a:b:c$。在考卷中，如果系数是变量，务必先化简为标准型。
    \item \textbf{系数倍数}：有时题目给出 $2\vec{OA} + 2\vec{OB} + 2\vec{OC} = 6\vec{OG}$，其本质依然是重心结论。
\end{itemize}
\end{trapbox}
