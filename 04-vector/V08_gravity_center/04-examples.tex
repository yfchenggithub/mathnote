\begin{examplebox}
\textbf{例题：} 若 $G$ 是 $\triangle ABC$ 的重心，$P$ 是平面内一点，满足 $\vec{PA} + \vec{PB} + \vec{PC} = 3\vec{PG}$，请判定点 $G$ 的位置。

\textit{解析：} 
由重心引申结论可知，对于任意点 $P$：
\[ \vec{PA} + \vec{PB} + \vec{PC} = (\vec{PG}+\vec{GA}) + (\vec{PG}+\vec{GB}) + (\vec{PG}+\vec{GC}) \]
\[ = 3\vec{PG} + (\vec{GA} + \vec{GB} + \vec{GC}) \]
因为 $G$ 为重心，$\vec{GA} + \vec{GB} + \vec{GC} = \vec{0}$。
所以等式恒成立。这验证了重心作为"向量均值点"的唯一性。
\end{examplebox}
