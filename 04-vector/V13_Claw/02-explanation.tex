\textbf{定理深度解读}：
\begin{itemize}
    \item \textbf{"爪子"形象}：该定理涉及四个点，从点 $A$ 出发引出三条向量，形状像鸟类的爪子，故名"爪子模型"。
    \item \textbf{系数的意义}：
        \begin{itemize}
            \item 当 $0 < \lambda < 1$ 时，点 $P$ 在线段 $MN$ 内部；
            \item 当 $\lambda > 1$ 或 $\lambda < 0$ 时，点 $P$ 在线段 $MN$ 的延长线或反向延长线上；
            \item $\lambda$ 的本质是分点比：$\lambda = \frac{PN}{MN}$（注意向量方向）。
        \end{itemize}
    \item \textbf{基底独立性}：该结论与起点 $A$ 的位置无关，这使得我们在解题时可以灵活选取最方便的观测点作为起点。
\end{itemize}
\end{explanationbox}
