\begin{proofbox}
\textbf{爪子定理证明}：
\begin{enumerate}
    \item \textbf{充分性}：若 $\vec{AP} = \lambda \vec{AM} + (1 - \lambda) \vec{AN}$，
    则 $\vec{AP} - \vec{AN} = \lambda (\vec{AM} - \vec{AN})$。
    由向量减法定义知：$\vec{NP} = \lambda \vec{NM}$。
    由于 $\vec{NP}$ 与 $\vec{NM}$ 共线且有公共点 $N$，故 $M, P, N$ 三点共线。
    
    \item \textbf{必要性}：若 $M, P, N$ 三点共线，则向量 $\vec{NP}$ 与 $\vec{NM}$ 共线。
    故存在实数 $\lambda$，使得 $\vec{NP} = \lambda \vec{NM}$。
    代入起点 $A$ 拆分：$\vec{AP} - \vec{AN} = \lambda (\vec{AM} - \vec{AN})$。
    整理得：$\vec{AP} = \lambda \vec{AM} + (1 - \lambda) \vec{AN}$。
\end{enumerate}
\end{proofbox}
