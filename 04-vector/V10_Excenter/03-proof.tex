\begin{proofbox}
\textbf{旁心基准等式的证明（奔驰定理推广）}：
\begin{enumerate}
    \item 设 $O$ 为 $\angle A$ 的旁心，由奔驰定理的面积权重公式：
    $S_{\triangle BOC} \cdot \vec{OA} + S_{\triangle AOC} \cdot \vec{OB} + S_{\triangle AOB} \cdot \vec{OC} = \vec{0}$。
    \item 注意点 $O$ 在 $\triangle ABC$ 外部，且与顶点 $A$ 位于直线 $BC$ 的异侧，此时 $S_{\triangle BOC}$ 需取有向面积的负值。
    \item 设旁切圆半径为 $r_a$，则：
    $S_{\triangle AOC} = \frac{1}{2} b r_a, \quad S_{\triangle AOB} = \frac{1}{2} c r_a$。
    $S_{\triangle BOC} = S_{\triangle AOC} + S_{\triangle AOB} - S_{\triangle ABC}$。
    \item 经几何推导，其面积比满足 $|S_{\triangle BOC}| : |S_{\triangle AOC}| : |S_{\triangle AOB}| = a : b : c$。
    \item 考虑到有向面积符号，对应系数比例为 $-a : b : c$。
    \item 故 $-a\vec{OA} + b\vec{OB} + c\vec{OC} = \vec{0}$ 成立。
\end{enumerate}
\end{proofbox}
