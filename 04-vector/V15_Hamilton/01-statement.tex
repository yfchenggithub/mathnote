\begin{statementbox}
设 $\triangle ABC$ 的外心为 $O$，垂心为 $H$，外接圆半径为 $R$，边长为 $a, b, c$，则：
\begin{enumerate}
    \item \textbf{核心等式}：$\vec{OH} = \vec{OA} + \vec{OB} + \vec{OC}$；
    \item \textbf{顶点表示}：$\vec{AH} = \vec{OB} + \vec{OC}$（同理对 $B, C$ 顶点亦成立）；
    \item \textbf{模长平方结论}：$|\vec{OH}|^2 = 9R^2 - (a^2 + b^2 + c^2)$；
    \item \textbf{距离关系}：$|\vec{AH}|^2 = 4R^2 - a^2$。
\end{enumerate}
\end{statementbox}
