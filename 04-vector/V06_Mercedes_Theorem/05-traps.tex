\begin{itemize}
    \item \textbf{顶点对应陷阱：} 务必记住 $S_A$ 对应的是以 $BC$ 为底边的三角形面积，即点 $O$ "对面"的面积。
    \item \textbf{符号陷阱：} 向量必须全部以点 $O$ 为起点（或全部以点 $O$ 为终点）。如果出现 $\vec{AO} = 2\vec{OB} + \dots$，需先转化为 $-\vec{OA}$。
    \item \textbf{五心判定：} 结合奔驰定理可以快速判断四心：
    \begin{itemize}
        \item 重心：$1:1:1$
        \item 垂心：$\tan A : \tan B : \tan C$
        \item 外心：$\sin 2A : \sin 2B : \sin 2C$
        \item 内心：$a : b : c$
    \end{itemize}
\end{itemize}
\end{trapbox}
