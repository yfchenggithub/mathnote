\begin{proofbox}
利用三角形面积比与线段比的转化：
\begin{enumerate}
    \item 延长 $AO$ 交 $BC$ 于点 $D$。根据面积比性质，有：
    \[ \frac{BD}{DC} = \frac{S_{\triangle ABD}}{S_{\triangle ACD}} = \frac{S_{\triangle OBD}}{S_{\triangle OCD}} = \frac{S_{\triangle ABD}-S_{\triangle OBD}}{S_{\triangle ACD}-S_{\triangle OCD}} = \frac{S_C}{S_B} \]
    \item 由分点坐标公式得：$\vec{OD} = \frac{S_B \vec{OB} + S_C \vec{OC}}{S_B + S_C}$。
    \item 又因为 $\frac{AO}{OD} = \frac{S_{\triangle ABO}}{S_{\triangle BDO}} = \frac{S_{\triangle ACO}}{S_{\triangle CDO}} = \frac{S_B+S_C}{S_A}$。
    \item 将 $\vec{OD}$ 反向表示：$\vec{OA} = -\frac{S_B+S_C}{S_A} \vec{OD}$。
    \item 代入化简即证：$S_A \vec{OA} + S_B \vec{OB} + S_C \vec{OC} = \vec{0}$。
\end{enumerate}
\end{proofbox}
