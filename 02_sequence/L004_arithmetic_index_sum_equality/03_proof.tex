\begin{proofbox}
\textbf{\textcolor{CProof}{思路提示}}

利用等差数列通项公式$a_n = a_1 + (n-1)d$，将$a_m+a_n$与$a_p+a_q$展开，比较下标和即可。

\medskip
\textbf{\textcolor{CProof}{正式推导}}
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.55em]
    \item[\textbf{步骤一（设公差）：}] 设等差数列$\{a_n\}$的公差为$d$，首项为$a_1$。
    \[
    a_n = a_1 + (n-1)d,\quad n\ge 1.
    \]
    \par\textbf{理由：} 通项公式是等差数列的基本表达。

    \item[\textbf{步骤二（展开左边）：}] 将$a_m+a_n$按通项展开。
    \[
    a_m+a_n = [a_1+(m-1)d] + [a_1+(n-1)d] = 2a_1 + (m+n-2)d.
    \]
    \par\textbf{理由：} 合并同类项，注意首项出现两次。

    \item[\textbf{步骤三（展开右边）：}] 同理展开$a_p+a_q$。
    \[
    a_p+a_q = [a_1+(p-1)d] + [a_1+(q-1)d] = 2a_1 + (p+q-2)d.
    \]
    \par\textbf{理由：} 同样的合并操作。

    \item[\textbf{步骤四（利用条件）：}] 条件$m+n=p+q$代入，得
    \[
    a_m+a_n = 2a_1 + (m+n-2)d = 2a_1 + (p+q-2)d = a_p+a_q.
    \]
    \par\textbf{理由：} 下标和相等导致一次项系数相等，因此两边相等。
\end{description}

\medskip
\textbf{\textcolor{CProof}{结论回扣}}

\textbf{证明结束：} 从通项出发，利用下标和相等这一代数条件，直接得出和相等。该证明无漏洞，结果对任意等差数列恒成立。
\end{proofbox}
