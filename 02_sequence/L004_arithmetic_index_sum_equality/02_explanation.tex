\begin{explanationbox}
\textbf{\textcolor{CExplanation}{一句话直觉}}

等差数列中，每一项都由首项和公差线性决定，下标和相等意味着“对称位置”的项数偏移相互抵消，因此和保持不变。

\medskip
\textbf{\textcolor{CExplanation}{核心拆解}}
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.45em]
    \item[\textbf{要点一（下标和相等）：}] 条件的核心是下标之间的算术关系，与项的具体位置无关。
    \item[\textbf{要点二（通项展开）：}] 利用$a_n=a_1+(n-1)d$，可将每一项展开为关于$n$的线性式，和式中的常数项与一次项恰好由下标和决定。
\end{description}

\medskip
\textbf{\textcolor{CExplanation}{几何本质（数轴对称）}}

在数轴上，将$n$视为坐标，等差数列的点$(n,a_n)$共线。下标和相等对应两个点的中点在直线上，其函数值之和等于中点函数值的两倍（线性函数的性质）。

\medskip
\textbf{\textcolor{CExplanation}{代数意义（系数对称）}}

通项公式$a_n=a_1+(n-1)d$中，$a_n$关于$n$是一次函数。一次函数满足：当自变量和相等时，函数值和相等。这是线性函数的基本性质。

\medskip
\textbf{\textcolor{CExplanation}{考点价值（快速求和）}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{考法一（直接求值）：} 已知等差数列中两项，求另两项和，无需计算公差。
    \item \textbf{考法二（构造配对）：} 在数列求和或证明中，常利用该性质将项配对简化。
\end{itemize}

\medskip
\textbf{\textcolor{CExplanation}{顿悟点}}

\textbf{下标和是“连通的钥匙”：} 只要下标和相等，不论项在数列中的位置多么分散，和都相同。这一结论可类比一次函数的“中点坐标”性质。

\medskip
\textbf{\textcolor{CExplanation}{使用场景}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{场景一（已知部分项求其他）：} 如已知$a_3$和$a_7$，直接得$a_2+a_8$或$a_1+a_9$等。
    \item \textbf{场景二（简化方程组）：} 在解等差数列的方程组时，利用该性质可避免解出公差。
\end{itemize}
\end{explanationbox}
