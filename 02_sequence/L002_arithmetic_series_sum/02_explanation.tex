\begin{explanationbox}
\textbf{\textcolor{CExplanation}{一句话直觉}}
等差数列求和如同计算等间距排列的梯形面积。

\medskip
\textbf{\textcolor{CExplanation}{核心拆解}}
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.45em]
    \item[\textbf{要点一（倒序相加）：}] 将原数列与倒序数列对应项相加，每对和相等。
    \item[\textbf{要点二（首尾平均）：}] 每对和均为 $a_1+a_n$，共 $n$ 对，总和除以 $2$ 即得 $S_n$。
\end{description}

\medskip
\textbf{\textcolor{CExplanation}{几何本质（梯形面积模型）}}
数列各项可视为等距排列的“高度”，求和等于以 $n$ 为宽、以 $(a_1+a_n)/2$ 为高的矩形面积，或如梯形面积（上底 + 下底）$\times$ 高 / 2。

\medskip
\textbf{\textcolor{CExplanation}{代数意义（二次函数型）}}
$S_n$ 是关于 $n$ 的二次函数 $S_n = \frac{d}{2}n^2 + \bigl(a_1 - \frac{d}{2}\bigr)n$，常数项为零，其图像过原点，对称轴与公差有关。

\medskip
\textbf{\textcolor{CExplanation}{考点价值（灵活求参数）}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{考法一（已知首尾求项数）：} 直接代入 $S_n = n(a_1+a_n)/2$，解方程求 $n$。
    \item \textbf{考法二（由片段和列方程）：} 利用 $S_n$ 的两种形式建立关于 $a_1,d$ 的方程组。
\end{itemize}

\medskip
\textbf{\textcolor{CExplanation}{顿悟点}}
\textbf{梯形面积公式与求和公式完全对应：上底 $a_1$，下底 $a_n$，高 $n$，面积就是前 $n$ 项和。}

\medskip
\textbf{\textcolor{CExplanation}{使用场景}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{场景一（已知首末项直接求和）：} 如题目给出 $a_1,a_n,n$，选用 $S_n = \frac{n(a_1+a_n)}{2}$。
    \item \textbf{场景二（已知首项公差求部分和）：} 如已知 $a_1,d,n$，选用 $S_n = na_1 + \frac{n(n-1)}{2}d$。
\end{itemize}
\end{explanationbox}
