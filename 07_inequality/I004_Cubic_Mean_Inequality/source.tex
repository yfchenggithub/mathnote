\subsection{三项立方和不等式}

\Conclusion

对于任意非负实数 \(a, b, c \in [0, +\infty)\)，有以下核心不等式结论：

\begin{enumerate}
    \item \textbf{三项均值不等式}：
    \[ \frac{a^3 + b^3 + c^3}{3} \geq abc; \]

    \item \textbf{常用形式}：
    \[ a^3 + b^3 + c^3 \geq 3abc; \]

    \item \textbf{等号成立条件}：当且仅当 \(a = b = c\) 时，等号成立。
\end{enumerate}
