\begin{proofbox}
\textbf{\textcolor{CProof}{思路提示}}

通过计算正弦函数的二阶导数证明其在 $(0, \pi)$ 上为凹函数，然后利用凹函数的定义导出不等式，最后在三角形中应用。

\medskip
\textbf{\textcolor{CProof}{正式推导}}
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.55em]
    \item[\textbf{步骤一（计算导数）：}] 对 $f(x) = \sin x$ 求导。
    \[
    f'(x) = \cos x, \quad f''(x) = -\sin x.
    \]
    \par\textbf{理由：} 正弦函数的一阶导为余弦，二阶导为负正弦。

    \item[\textbf{步骤二（判断凹性）：}] 在区间 $(0, \pi)$ 上，有 $\sin x > 0$，故 $f''(x) = -\sin x < 0$。
    \[
    f''(x) < 0 \quad \text{对任意 } x \in (0, \pi).
    \]
    \par\textbf{理由：} 二阶导为负是函数为凹的充分条件。

    \item[\textbf{步骤三（应用凹性）：}] 对于凹函数 $f$，有 $f\left(\frac{x+y}{2}\right) \geq \frac{f(x)+f(y)}{2}$。
    \[
    \sin\!\left(\frac{x+y}{2}\right) \geq \frac{\sin x + \sin y}{2}.
    \]
    \par\textbf{理由：} 将 $f(x)=\sin x$ 代入凹函数不等式。

    \item[\textbf{步骤四（三角形推广）：}] 设 $A, B, C$ 为三角形内角，则 $A+B+C=\pi$，且 $A, B, C \in (0, \pi)$。
    \[
    \frac{\sin A + \sin B + \sin C}{3} \leq \sin\!\left(\frac{A+B+C}{3}\right) = \sin\frac{\pi}{3} = \frac{\sqrt{3}}{2}.
    \]
    \par\textbf{理由：} 对三个点应用凹函数不等式，并利用三角形内角和为 $\pi$。
\end{description}

\medskip
\textbf{\textcolor{CProof}{结论回扣}}

综上，正弦函数在 $(0, \pi)$ 上的凹性导致不等式 $\frac{\sin x + \sin y}{2} \leq \sin\left(\frac{x+y}{2}\right)$，在三角形中进一步得到 $\frac{\sin A + \sin B + \sin C}{3} \leq \frac{\sqrt{3}}{2}$，取等条件分别为 $x=y$ 和 $A=B=C$。
\end{proofbox}
