\begin{summarybox}
\textbf{\textcolor{CSummary}{一句话核心}}
\textbf{凸函数“中点值 ≤ 平均值”，凹函数相反；该结论可推广到加权平均，是证明许多重要不等式的有力工具。}

\medskip
\textbf{\textcolor{CSummary}{使用条件}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{条件 1（函数性质）：} 函数在区间 $I$ 上连续且二阶可导（至少已知凸凹性）。
    \item \textbf{条件 2（凸凹性明）：} 明确二阶导符号（$f''>0$ 凸，$f''<0$ 凹）或已知凸凹性。
    \item \textbf{条件 3（点属区间）：} 所有涉及的自变量 $x_i$ 均属于定义区间 $I$。
\end{itemize}

\medskip
\textbf{\textcolor{CSummary}{关键提醒}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{易错点（方向判断）：} 务必根据凸凹性确定不等号方向，可借图像记忆。
    \item \textbf{检查项（取等条件）：} 使用后检查等号成立条件，通常为所有自变量相等。
\end{itemize}
\end{summarybox}
