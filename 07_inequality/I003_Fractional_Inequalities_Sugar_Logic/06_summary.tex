\begin{summarybox}
\textbf{\textcolor{CSummary}{一句话核心}}
\textbf{糖水不等式描述了分式分子分母同加减同数时，分式值的变化规律：加糖增大，减糖减小。}

\medskip
\textbf{\textcolor{CSummary}{使用条件}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{条件 1（全为正数）：} $a, b, m > 0$。
    \item \textbf{条件 2（原式小于1）：} $a < b$（保证原分式 $\frac{a}{b} < 1$）。
    \item \textbf{条件 3（减糖额外）：} 应用减糖时需 $b > m$。
\end{itemize}

\medskip
\textbf{\textcolor{CSummary}{关键提醒}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{易错点（符号检查）：} 务必先检查所有变量为正，否则结论可能失效。
    \item \textbf{检查项（倒数关联）：} 推广结论可由加糖原理取倒数导出，减少记忆量。
\end{itemize}
\end{summarybox}
