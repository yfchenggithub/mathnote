\section*{1.42 三元循环平方分式不等式}
对于任意正实数 \( x, y, z > 0 \)：
\begin{enumerate}
    \item 核心形式：\( \frac{x^2}{y} + \frac{y^2}{z} + \frac{z^2}{x} \geq x + y + z \)；
    \item 链式关系：\( \frac{x^2}{y} + \frac{y^2}{z} + \frac{z^2}{x} \geq \frac{x+y+z}{3} \cdot 3 = x+y+z \)；
    \item 等号成立条件：当且仅当 \( x = y = z \)。
\end{enumerate}
