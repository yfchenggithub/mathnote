\begin{examplebox}
【例1（基础应用）】
已知函数 $g(x)=x^2-2x$，若 $1 < g(x) < 3$，试用乘积形式表达这个条件。

\textbf{解：}
这里 $N=1, M=3, f(x)=g(x)=x^2-2x$。
直接套用结论中的乘积形式：
\[
[g(x)-3][g(x)-1] < 0
\]
即
\[
[(x^2-2x)-3][(x^2-2x)-1] < 0
\]
化简得 $(x^2-2x-3)(x^2-2x-1) < 0$。

【例2（需要转化）】
若对于所有实数 $x$，都有 $\left| h(x) - 2 \right| < 1$，请用分式形式写出 $h(x)$ 满足的不等式。

\textbf{解：}
绝对值形式 $\left| h(x) - 2 \right| < 1$ 对应结论中的 $\left| f(x) - \frac{M+N}{2} \right| < \frac{M-N}{2}$。
对比可知：中点 $\frac{M+N}{2}=2$，半径 $\frac{M-N}{2}=1$。
解得 $M=3, N=1$。
因此原不等式等价于 $1 < h(x) < 3$。
再套用分式形式：
\[
\frac{h(x)-1}{3-h(x)} > 0
\]
\end{examplebox}
