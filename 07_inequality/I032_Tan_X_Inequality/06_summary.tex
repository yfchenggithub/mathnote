\begin{summarybox}
\textbf{\textcolor{CSummary}{一句话核心}}
\textbf{在锐角范围内，正弦与正切的和恒大于该角的两倍，其算术平均值恒大于该角本身。}

\medskip
\textbf{\textcolor{CSummary}{使用条件}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{条件 1（角为锐角）：} 自变量 $x$ 必须满足 $0 < x < \frac{\pi}{2}$。
    \item \textbf{条件 2（整体应用）：} 结论适用于 $\sin x$ 与 $\tan x$ 的“和”或“算术平均”形式，单独一项时不成立。
\end{itemize}

\medskip
\textbf{\textcolor{CSummary}{关键提醒}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{易错点（区间内严格）：} 在 $(0, \pi/2)$ 内不等式严格大于，无等号。等号仅在 $x=0$（端点）时成立。
    \item \textbf{检查项（结论选择）：} 根据题目需要，选择使用“和大于 $2x$”还是“平均值大于 $x$”，前者更强。
    \item \textbf{关联知识（放缩链条）：} 可与 $\sin x < x < \tan x$ 结合，形成完整的放缩链条：$\sin x < x < \frac{\sin x + \tan x}{2} < \frac{2x}{?}$ 注意排序。
\end{itemize}
\end{summarybox}
