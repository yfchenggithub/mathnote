\begin{summarybox}
\textbf{一句话核心}

\textbf{两组实数平方和的乘积，永远不小于它们对应项乘积之和的平方；等号当且仅当两组数成比例时取得。}

\medskip
\textbf{使用条件}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{条件 1：} 涉及两组实数的平方和与乘积和的混合表达式。
    \item \textbf{条件 2：} 目标为证明某个不等式，或求某个表达式的最值（尤其是线性式在二次约束下的最值）。
    \item \textbf{条件 3：} 两组数的项数必须相同，但可以灵活构造和选取这些数。
\end{itemize}

\medskip
\textbf{关键提醒}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{易错点：} 忽略取等条件与题目实际约束的兼容性，导致最值不可达。
    \item \textbf{检查项：} 应用前，明确构造的 $a_i$ 和 $b_i$ 分别是什么；应用后，务必验证取等条件是否能成立。
    \item \textbf{技巧：} 当分母为和的形式时，常利用 $\frac{x^2}{y}$ 的结构，配合柯西不等式将分子化为平方和。
    \item \textbf{关联：} 向量形式 $|\vec{a}\cdot\vec{b}| \leq |\vec{a}|\,|\vec{b}|$ 提供了直观的几何解释（夹角余弦的绝对值不超过1）。
\end{itemize}
\end{summarybox}
