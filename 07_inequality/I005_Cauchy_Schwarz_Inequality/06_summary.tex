\begin{summarybox}
\textbf{一句话核心}

\textbf{柯西不等式通过平方和非负性，建立了两组实数平方和乘积与它们对应乘积和平方的不等关系。}

\medskip
\textbf{使用条件}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{条件 1（实数序列）：} 涉及的两组数 $a_i, b_i$ 均为实数。
    \item \textbf{条件 2（平方和存在）：} 不等式两边的平方和 $\sum a_i^2$ 和 $\sum b_i^2$ 有意义（通常为有限和）。
\end{itemize}

\medskip
\textbf{关键提醒}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{易错点（验证取等）：} 使用柯西不等式求最值时，必须验证等号成立条件（两组数成比例或一组全为零）。
    \item \textbf{检查项（构造匹配）：} 应用前需将问题化为标准形式，确保构造的两组数能使乘积和与目标式一致。
\end{itemize}
\end{summarybox}
