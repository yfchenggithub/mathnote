\begin{explanationbox}
\textbf{\textcolor{CExplanation}{一句话直觉}}
正数倒数的均值关系，本质是柯西不等式或均值不等式的直接推论。

\medskip
\textbf{\textcolor{CExplanation}{核心拆解}}
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.45em]
    \item[\textbf{要点一（均值变形）：}] 从基本不等式 $(x+y)^2 \ge 4xy$ 出发，可立即推出倒数和形式。
    \item[\textbf{要点二（柯西特例）：}] 权方和形式是柯西不等式的二维特例 $(a_1^2/b_1 + a_2^2/b_2)(b_1+b_2) \ge (a_1+a_2)^2$。
    \item[\textbf{要点三（形式等价）：}] 倒数和形式与和积形式仅是移项区别，本质相同。
\end{description}

\medskip
\textbf{\textcolor{CExplanation}{几何本质（倒数和关系）}}
该不等式描述了正数与其倒数的一种“调和”关系。
当两个正数相等时，它们的倒数也相等，此时“倒数和”与“和的倒数”之间达到最紧的比值 $4$。

\medskip
\textbf{\textcolor{CExplanation}{代数意义（优化工具）}}
这是处理分式不等式的重要工具。
尤其当问题中出现 $\frac{1}{x}+\frac{1}{y}$ 与 $x+y$ 的组合时，可直接建立桥梁。
权方和形式则用于将平方项与线性项“分配”到分母上，实现放缩目标。

\medskip
\textbf{\textcolor{CExplanation}{考点价值（分式放缩）}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{考法一（直接应用）：} 在条件不等式证明中，直接使用结论建立不等关系。
    \item \textbf{考法二（变形构造）：} 需先对目标式进行恒等变形，再凑出适用形式。
    \item \textbf{考法三（权方和放缩）：} 处理含平方项的分式最值或证明问题。
\end{itemize}

\medskip
\textbf{\textcolor{CExplanation}{顿悟点}}
\textbf{当看到 $\frac{1}{x}+\frac{1}{y}$ 与 $x+y$ 同时出现时，要立刻联想到该不等式链。}
\textbf{权方和形式本质是“平方平均≥算术平均”在带权情形下的体现。}

\medskip
\textbf{\textcolor{CExplanation}{使用场景}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{场景一（分式最值）：} 求 $\frac{1}{x}+\frac{1}{y}$ 在 $x+y$ 定值下的最小值。
    \item \textbf{场景二（不等式证明）：} 在复杂分式不等式中作为中间放缩步骤。
    \item \textbf{场景三（系数匹配）：} 权方和形式用于匹配系数，将分散项合并。
\end{itemize}
\end{explanationbox}
