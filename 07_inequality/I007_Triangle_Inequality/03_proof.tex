\begin{proofbox}
\textbf{\textcolor{CProof}{思路提示}}

通过平方去掉绝对值，将不等式转化为平方的非负性证明，然后讨论等号成立条件。

\medskip
\textbf{\textcolor{CProof}{正式推导}}
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.55em]
    \item[\textbf{步骤一（平方变形）：}] 考虑 $|a+b|^2$ 与 $(|a|+|b|)^2$ 的比较。
    \[
    |a+b|^2 = (a+b)^2 = a^2 + 2ab + b^2.
    \]
    \par\textbf{理由：} 绝对值的平方等于原数的平方。

    \item[\textbf{步骤二（展开右边）：}] 计算 $(|a|+|b|)^2$。
    \[
    (|a|+|b|)^2 = |a|^2 + 2|a||b| + |b|^2 = a^2 + 2|ab| + b^2.
    \]
    \par\textbf{理由：} $|a|^2 = a^2$，且 $|a||b| = |ab|$。

    \item[\textbf{步骤三（比较差值）：}] 比较两者之差。
    \[
    (|a|+|b|)^2 - |a+b|^2 = (a^2 + 2|ab| + b^2) - (a^2 + 2ab + b^2) = 2(|ab| - ab).
    \]
    \par\textbf{理由：} 相减后化简。

    \item[\textbf{步骤四（判断非负）：}] 由于 $|ab| \geq ab$ 恒成立，因此 $|ab| - ab \geq 0$。
    \[
    (|a|+|b|)^2 - |a+b|^2 = 2(|ab| - ab) \geq 0.
    \]
    \par\textbf{理由：} 绝对值的定义保证了 $|x| \geq x$ 对所有实数 $x$ 成立。

    \item[\textbf{步骤五（得出右边不等式）：}] 由平方非负，得 $|a+b|^2 \leq (|a|+|b|)^2$，开方后得到 $|a+b| \leq |a|+|b|$。
    \[
    |a+b| \leq |a| + |b|.
    \]
    \par\textbf{理由：} 平方函数在非负数上单调递增。

    \item[\textbf{步骤六（左边不等式证明）：}] 利用右边不等式，将 $a$ 替换为 $a-b$，可得 $|a| \leq |a-b| + |b|$，从而 $|a| - |b| \leq |a-b|$。同理 $|b| - |a| \leq |a-b|$，所以 $\bigl|\,|a| - |b|\,\bigr| \leq |a-b|$。再令 $b$ 为 $-b$，即得 $\bigl|\,|a| - |b|\,\bigr| \leq |a+b|$。
    \[
    \bigl|\,|a| - |b|\,\bigr| \leq |a+b|.
    \]
    \par\textbf{理由：} 通过变量替换和对称性推导。

    \item[\textbf{步骤七（等号条件）：}] 右边等号成立当 $|ab| = ab$，即 $ab \geq 0$；左边等号成立需在推导中取等，即 $ab \leq 0$。
    \par\textbf{理由：} 从差值 $2(|ab|-ab)$ 为零可得 $ab \geq 0$；左边等号当 $a$ 与 $b$ 异号时成立。
\end{description}

\medskip
\textbf{\textcolor{CProof}{结论回扣}}

综上所述，我们证明了对于任意实数 $a, b$，有 $\bigl|\,|a| - |b|\,\bigr| \leq |a+b| \leq |a| + |b|$，并明确了等号成立的条件。
\end{proofbox}
