\begin{explanationbox}
\textbf{\textcolor{CExplanation}{一句话直觉}}

绝对值三角不等式是“三角形两边之和大于第三边”在数轴上的代数版本，它限制了实数和的绝对值范围。

\medskip
\textbf{\textcolor{CExplanation}{核心拆解}}
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.45em]
    \item[\textbf{要点一（下界控制）：}] 左边不等式 $\bigl|\,|a| - |b|\,\bigr| \leq |a + b|$ 保证了两数和的绝对值不小于它们绝对值之差的绝对值。
    \item[\textbf{要点二（上界控制）：}] 右边不等式 $|a + b| \leq |a| + |b|$ 保证了两数和的绝对值不超过它们绝对值之和。
\end{description}

\medskip
\textbf{\textcolor{CExplanation}{几何本质（距离关系）}}

在数轴上，$|a|$ 表示 $a$ 到原点的距离。$|a+b|$ 可以视为 $a$ 与 $-b$ 的距离，而 $|a|+|b|$ 是两段距离之和，$\bigl|\,|a| - |b|\,\bigr|$ 是两段距离之差的绝对值。不等式反映了“两点间直线段最短”的思想。

\medskip
\textbf{\textcolor{CExplanation}{代数意义（运算约束）}}

绝对值运算不满足线性，但此不等式给出了其与加法运算的一种线性约束：和的绝对值被绝对值之和控制，且受差值绝对值限制。

\medskip
\textbf{\textcolor{CExplanation}{考点价值（不等式证明）}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{考法一（直接应用）：} 在证明题中直接使用该不等式放缩。
    \item \textbf{考法二（最值求解）：} 利用不等式求含绝对值的表达式的最值或范围。
\end{itemize}

\medskip
\textbf{\textcolor{CExplanation}{顿悟点}}

等号成立条件与实数的符号密切相关：当 $a$ 与 $b$ 异号时，左边等号成立；当 $a$ 与 $b$ 同号时，右边等号成立。这揭示了绝对值运算与符号的内在联系。

\medskip
\textbf{\textcolor{CExplanation}{使用场景}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{场景一（比较大小）：} 需要比较 $|a+b|$ 与 $|a|+|b|$ 或 $\bigl|\,|a| - |b|\,\bigr|$ 的大小时。
    \item \textbf{场景二（估计范围）：} 当已知 $|a|, |b|$ 的范围时，估计 $|a+b|$ 的可能范围。
\end{itemize}
\end{explanationbox}
