\begin{trapbox}
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.35em]
    \item[\textbf{\textcolor{CTrap}{易错点一（忽视正数条件）}}] 只记住 $G<L<A$ 的形式，而忽略 $x_1, x_2$ 必须是正实数的前提条件。
    \item[\textbf{\textcolor{CTrap}{正确理解（正数是前提）：}}] 对数 $\ln x$ 的定义域是 $x>0$，因此 $x_1, x_2$ 必须为正实数，结论才成立。若出现负数或零，不等式无意义。
    \item[\textbf{\textcolor{CTrap}{错因分析（定义域遗忘）：}}] 学生常因机械记忆不等式形式，而忽略对数函数本身的定义域限制，导致在非正数范围误用结论。
\end{description}

\medskip
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.35em]
    \item[\textbf{\textcolor{CTrap}{易错点二（方向记反）}}] 将不等式链 $G < L < A$ 的方向记反，例如误以为 $L$ 最大或最小。
    \item[\textbf{\textcolor{CTrap}{正确理解（L居中）：}}] 对数平均值 $L$ 是介于几何平均 $G$ 和算术平均 $A$ 之间的，并且 $G$ 最小，$A$ 最大。可以简记为“小几中对大算”。
    \item[\textbf{\textcolor{CTrap}{错因分析（直观混淆）：}}] 学生可能误以为“对数”复杂所以值较大，或混淆了 $\ln x$ 增长缓慢的性质与平均值大小的关系。
\end{description}

\medskip
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.35em]
    \item[\textbf{\textcolor{CTrap}{易错点三（等号条件误判）}}] 认为当 $x_1 = x_2$ 时，不等式 $G < L < A$ 仍然成立（即不等号仍严格成立）。
    \item[\textbf{\textcolor{CTrap}{正确理解（相等时取等）：}}] 当 $x_1 = x_2$ 时，对数平均值 $L$ 定义为极限值（洛必达法则可得 $L = x_1$），此时 $G = L = A$，不等式变为等式。
    \item[\textbf{\textcolor{CTrap}{错因分析（定义扩展不清）：}}] 结论的原始条件是不相等，但学生常忽略相等时的连续性情况，误以为不等式永远严格成立。
\end{description}
\end{trapbox}
