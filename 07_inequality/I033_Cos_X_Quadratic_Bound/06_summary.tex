\begin{summarybox}
\textbf{\textcolor{CSummary}{一句话核心}}
\textbf{对数平均值不等式 $G < L < A$ 是连接几何平均与算术平均的重要桥梁，其核心是将对数差用代数式进行缩放。}

\medskip
\textbf{\textcolor{CSummary}{使用条件}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{条件 1（正实数）：} 涉及的两个数 $x_1, x_2$ 必须为正实数，因为需要取对数。
    \item \textbf{条件 2（不相等）：} 两个数不相等，此时不等式严格成立；若相等，则三个平均值相等。
\end{itemize}

\medskip
\textbf{\textcolor{CSummary}{关键提醒}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{易错点（方向记牢）：} 务必记住 $\sqrt{x_1 x_2}$ 最小，$\dfrac{x_1+x_2}{2}$ 最大，对数平均值 $L$ 在中间。
    \item \textbf{检查项（结构识别）：} 看到 $\dfrac{a-b}{\ln a - \ln b}$ 或类似结构，应立即联想到此不等式。
\end{itemize}
\end{summarybox}
