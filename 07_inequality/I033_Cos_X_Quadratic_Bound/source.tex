\section*{1.33 对数平均值不等式}
设 \( x_1,x_2 \) 为两个不相等的正实数，定义对数平均值 \( L(x_1,x_2)=\frac{x_1-x_2}{\ln x_1-\ln x_2} \)，则：
\begin{enumerate}
    \item 核心链式不等式：\( \sqrt{x_1x_2} < \frac{x_1-x_2}{\ln x_1-\ln x_2} < \frac{x_1+x_2}{2} \)；
    \item 简记：\( G < L < A \)；
    \item 常用变形：
    \[
    \ln\frac{x_1}{x_2} > \frac{2(x_1-x_2)}{x_1+x_2},\qquad \ln\frac{x_1}{x_2} < \frac{x_1-x_2}{\sqrt{x_1x_2}}.
    \]
\end{enumerate}
