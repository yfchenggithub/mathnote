\begin{explanationbox}
\textbf{\textcolor{CExplanation}{一句话直觉}}
对于两个不相等的正数，对数平均值是几何平均与算术平均之间的一个“中间值”，它比几何平均大，但比算术平均小。

\medskip
\textbf{\textcolor{CExplanation}{核心拆解}}
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.45em]
    \item[\textbf{要点一（定义结构）：}] 对数平均值 $L$ 的分子是两数差，分母是两数对数差，比值恰好等于指数函数 $f(t)=e^t$ 在两点之间割线的斜率倒数，是一个“平均斜率”的概念。
    \item[\textbf{要点二（不等式链）：}] 最核心的结论是 $\sqrt{x_1 x_2} < L < \dfrac{x_1+x_2}{2}$，它连接了三个重要的平均值：几何平均 $G$、对数平均 $L$、算术平均 $A$。
\end{description}

\medskip
\textbf{\textcolor{CExplanation}{几何本质（指数函数斜率）}}
考虑函数 $y=\ln x$ 在点 $(x_1,\ln x_1)$ 与 $(x_2,\ln x_2)$ 之间的割线斜率，这个斜率恰好等于 $\dfrac{\ln x_1-\ln x_2}{x_1-x_2}$，它就是对数平均值 $L$ 的倒数。几何上，这个斜率介于函数在 $\sqrt{x_1x_2}$ 处的切线斜率（导数）与两端点连线的平均斜率之间，由此导出了 $G<L<A$。

\medskip
\textbf{\textcolor{CExplanation}{代数意义（对数差缩放）}}
等价形式 $\ln\dfrac{x_1}{x_2} > \dfrac{2(x_1-x_2)}{x_1+x_2}$ 和 $\ln\dfrac{x_1}{x_2} < \dfrac{x_1-x_2}{\sqrt{x_1x_2}}$ 将“对数差”用两种代数式进行上下界估计。左边用调和平均数相关的式子下界，右边用几何平均数相关的式子上界。

\medskip
\textbf{\textcolor{CExplanation}{考点价值（放缩工具）}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{考法一（比较大小）：} 在选择题中，直接比较 $\sqrt{ab}$、$\dfrac{a-b}{\ln a-\ln b}$、$\dfrac{a+b}{2}$ 三者的大小。
    \item \textbf{考法二（证明不等式）：} 作为中间放缩工具，用于证明涉及对数、指数与代数式混合的不等式，尤其在导数压轴题中常见。
\end{itemize}

\medskip
\textbf{\textcolor{CExplanation}{顿悟点}}
\textbf{遇到“两正数之差除以两数对数之差”的结构，或遇到需要联系几何平均与算术平均的题目，应立即联想到对数平均值不等式。}

\medskip
\textbf{\textcolor{CExplanation}{使用场景}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{场景一（直接应用）：} 题目中直接出现对数平均值 $L$，要求证明 $G<L<A$ 或其等价形式。
    \item \textbf{场景二（隐蔽转化）：} 题目给出形如 $\dfrac{f(a)-f(b)}{a-b}$ 的表达式，而 $f(x)=\ln x$ 或 $f(x)=e^x$，可转化为对数平均值进行放缩。
\end{itemize}
\end{explanationbox}
