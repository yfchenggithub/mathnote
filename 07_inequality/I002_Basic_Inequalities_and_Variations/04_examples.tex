\begin{examplebox}
【例1（基础应用）】
已知 $x>0$，求 $x + \frac{4}{x}$ 的最小值。

\textbf{解：}
1. 识别结构：$a=x$，$b=\frac{4}{x}$，都是正数。
2. 直接应用算术-几何平均不等式：
   \[
   x + \frac{4}{x} \ge 2\sqrt{x \cdot \frac{4}{x}}
   \]
3. 化简：
   \[
   x + \frac{4}{x} \ge 2\sqrt{4} = 4
   \]
4. 检查等号成立条件：当且仅当 $x = \frac{4}{x}$，即 $x^2=4$，$x=2$（$x>0$）。

👉 所以，当 $x=2$ 时，$x+\frac{4}{x}$ 取得最小值 $4$。

---

【例2（稍作变形）】
已知 $a, b>0$，且 $a+b=1$，求证：$ab \le \frac{1}{4}$。

\textbf{证：}
1. 已知条件是 $a+b$ 为定值 $1$，结论是关于 $ab$ 的最大值。
2. 想到变形形式：$ab \le ((a+b)/2)^2$。
3. 直接代入 $a+b=1$：
   \[
   ab \le \left(\frac{1}{2}\right)^2 = \frac{1}{4}
   \]
4. 等号成立当且仅当 $a=b$，结合 $a+b=1$，得 $a=b=\frac{1}{2}$。

👉 本题直接使用了结论的第三个变形，一步到位。
\end{examplebox}
