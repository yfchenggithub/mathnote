\begin{summarybox}
\textbf{一句话核心}

\textbf{基本不等式建立了两个非负数的和、积与平方和之间的不等关系，等号成立当且仅当两数相等，是求解最值和证明不等式的基石。}

\medskip
\textbf{使用条件}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{条件 1：} 使用 $a+b \ge 2\sqrt{ab}$ 或 $ab \le ((a+b)/2)^2$ 时，必须确保 $a \ge 0$ 且 $b \ge 0$。
    \item \textbf{条件 2：} 使用 $a^2+b^2 \ge 2ab$ 或 $a^2+b^2 \ge (a+b)^2/2$ 时，对任意实数 $a, b$ 都成立。
    \item \textbf{条件 3：} 用其求最值时，必须验证等号成立条件是否可达，且符合题目所有约束。
\end{itemize}

\medskip
\textbf{关键提醒}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{易错点：} 混淆不同形式的适用范围，特别是忽略非负条件。
    \item \textbf{检查项：} 1) 变量是否满足所用不等式的定义域；2) 等号成立条件能否取到；3) 所求最值是否合理（例如，最小值是否真的被取到）。
\end{itemize}
\end{summarybox}
