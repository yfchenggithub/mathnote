\begin{summarybox}
\textbf{一句话核心}

\textbf{基本不等式建立了两个正数的和、积、平方和之间的不等关系，是求解最值问题和证明不等式的基石。}

\medskip
\textbf{使用条件}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{条件1：} 涉及的不等式形式（如 $a+b$ 与 $\sqrt{ab}$）要求 $a,b$ 均为正实数。
    \item \textbf{条件2：} 当利用不等式求最值时，通常需要和为定值（求积的最大值）或积为定值（求和的最小值）。
\end{itemize}

\medskip
\textbf{关键提醒}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{易错点：} 忽略等号成立条件的验证，导致最值错误。
    \item \textbf{检查项：} 应用前先确认变量是否为正；求得最值后必须检查取等条件是否可达。
\end{itemize}
\end{summarybox}
