\begin{summarybox}
\textbf{一句话核心}

\textbf{对于两个正实数，它们的算术平均数总是不小于几何平均数，且等号成立当且仅当两数相等。}

\medskip
\textbf{使用条件}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{条件 1（正实数）：} 不等式中的 $a$ 和 $b$ 必须是正实数（$a>0,\,b>0$）。
    \item \textbf{条件 2（形式匹配）：} 需要将问题转化为和、积、平方和等标准形式，才能正确应用不等式。
\end{itemize}

\medskip
\textbf{关键提醒}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{易错点（取等验证）：} 利用不等式求最值时，必须检查等号成立条件是否满足，否则最值可能不可达。
    \item \textbf{检查项（前提确认）：} 使用前务必确认变量为正，避免在非正数情况下误用。
\end{itemize}
\end{summarybox}
