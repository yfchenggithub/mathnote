\begin{explanationbox}
\textbf{\textcolor{CExplanation}{一句话直觉}}

将平方项分配到分母为两数和的项上，其和至少为所有变量和的一半。

\medskip
\textbf{\textcolor{CExplanation}{核心拆解}}
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.45em]
    \item[\textbf{要点一（柯西-施瓦茨应用）：}] 使用柯西-施瓦茨不等式将分子平方与分母和关联。
    \item[\textbf{要点二（对称性简化）：}] 利用对称性，分母总和为 $2(a+b+c)$，简化计算。
\end{description}

\medskip
\textbf{\textcolor{CExplanation}{几何本质（平衡点下界）}}

在三维空间中，考虑向量分量，该不等式表示加权距离的下界，当各分量相等时达到平衡。

\medskip
\textbf{\textcolor{CExplanation}{代数意义（分式化整）}}

通过乘以分母和，将分式不等式转化为整式不等式，便于使用均值不等式或柯西-施瓦茨。

\medskip
\textbf{\textcolor{CExplanation}{考点价值（对称不等式）}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{考法一（直接证明）：} 作为经典不等式，要求直接证明。
    \item \textbf{考法二（求最值）：} 在给定条件下，求类似表达式的最小值。
\end{itemize}

\medskip
\textbf{\textcolor{CExplanation}{顿悟点}}

分母和恰好为 $2(a+b+c)$，从而与分子平方和匹配，导出下界 $\frac{a+b+c}{2}$。

\medskip
\textbf{\textcolor{CExplanation}{使用场景}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{场景一（对称不等式证明）：} 当题目中出现对称的分式平方和时。
    \item \textbf{场景二（最值问题）：} 求 $\frac{a^2}{b+c} + \frac{b^2}{c+a} + \frac{c^2}{a+b}$ 的最小值，已知 $a,b,c>0$。
\end{itemize}
\end{explanationbox}
