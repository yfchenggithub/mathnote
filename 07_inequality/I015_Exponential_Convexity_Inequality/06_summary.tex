\begin{summarybox}
\textbf{\textcolor{CSummary}{一句话核心}}

指数函数的凸性导致指数平均不小于平均的指数。

\medskip
\textbf{\textcolor{CSummary}{使用条件}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{条件 1（变量实数）：} $x_1, x_2, \dots, x_n$ 为任意实数。
    \item \textbf{条件 2（权重相等）：} 主结论中默认权重相等；加权形式需权重非负且和为1。
\end{itemize}

\medskip
\textbf{\textcolor{CSummary}{关键提醒}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{易错点（方向记反）：} 牢记凸函数导致 $\geq$，切勿反向。
    \item \textbf{检查项（取等条件）：} 检查等号是否在所有变量相等时取得。
\end{itemize}
\end{summarybox}
