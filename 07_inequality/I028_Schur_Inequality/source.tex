\section*{1.28 伯努利不等式}
对于实数 \( x \geq -1 \) 和指数 \( n \)：
\begin{enumerate}
    \item 若 \( n \geq 1 \) 或 \( n \leq 0 \)，则 \( (1+x)^n \geq 1+nx \)；
    \item 若 \( 0 \leq n \leq 1 \)，则 \( (1+x)^n \leq 1+nx \)；
    \item 等号条件：当且仅当 \( n=0,1 \) 或 \( x=0 \)；
    \item 常用变体：\( e^x \geq 1+x \)（\( x\in\mathbb{R} \)）。
\end{enumerate}
