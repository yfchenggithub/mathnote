\begin{explanationbox}
\textbf{\textcolor{CExplanation}{一句话直觉}}

在单位圆中，角度 $x$ 对应的正切线段长度大于弧长 $x$，而弧长大于正弦线段长度，从而形成不等式链。

\medskip
\textbf{\textcolor{CExplanation}{核心拆解}}
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.45em]
    \item[\textbf{要点一（区间分正负）：}] 不等式的符号取决于 $x$ 所在区间：非负时 $\tan x \geq x$，非正时 $\tan x \leq x$。
    \item[\textbf{要点二（不等号方向）：}] 在 $x>0$ 时，$\tan x$ 恒大于 $x$；在 $x<0$ 时，$\tan x$ 恒小于 $x$。
    \item[\textbf{要点三（严格夹逼）：}] 当 $x \in (0,\frac{\pi}{2})$ 时，有严格的上下界：$\sin x < x < \tan x$。
\end{description}

\medskip
\textbf{\textcolor{CExplanation}{几何本质（线段比较）}}

在单位圆中，设角度 $x$（弧度），则弧长 $=x$，正弦线长 $=\sin x$，正切线长 $=\tan x$。几何构造显示，对于 $x>0$，正切线最长，弧长次之，正弦线最短，即 $\tan x > x > \sin x$。

\medskip
\textbf{\textcolor{CExplanation}{代数意义（函数比较）}}

从函数角度看，在 $x=0$ 处，$\tan x$ 与 $x$ 有相同的线性项（导数均为 $1$），但 $\tan x$ 的泰勒展开为 $x + \frac{x^3}{3} + \cdots$，高阶项为正，故在 $x>0$ 时 $\tan x > x$；类似地，$\sin x$ 的展开为 $x - \frac{x^3}{6} + \cdots$，故 $\sin x < x$。

\medskip
\textbf{\textcolor{CExplanation}{考点价值（不等式证明）}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{考法一（直接比较）：} 在证明题中，直接利用 $\tan x \geq x$ 或 $\sin x < x$ 放缩，简化复杂不等式。
    \item \textbf{考法二（变形应用）：} 通过变量代换（如令 $t = \tan x$）将问题转化为这些标准形式，用于求极限或比较大小。
\end{itemize}

\medskip
\textbf{\textcolor{CExplanation}{顿悟点}}

当 $x$ 很小时，$\tan x \approx x + \frac{x^3}{3}$，因此 $\tan x$ 总是略大于 $x$（对于 $x>0$）；类似地，$\sin x \approx x - \frac{x^3}{6}$，所以 $\sin x$ 略小于 $x$。这解释了不等式链的严格性。

\medskip
\textbf{\textcolor{CExplanation}{使用场景}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{场景一（证明题）：} 证明如“$\tan 0.1 > 0.1$”或“$\sin 0.2 < 0.2$”类的不等式时，直接引用结论。
    \item \textbf{场景二（估值题）：} 给定 $x \in (0,\frac{\pi}{2})$，快速估算 $\sin x$ 或 $\tan x$ 的范围，例如由 $x < \tan x$ 知 $\tan x$ 的下界为 $x$。
\end{itemize}
\end{explanationbox}
