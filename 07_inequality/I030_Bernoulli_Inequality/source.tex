\section*{1.30 正切函数基础放缩}
对于任意实数 \( x \in [0,\frac{\pi}{2}) \)：
\begin{enumerate}
    \item 核心形式：\( \tan x \geq x \)；
    \item 对称形式：当 \( x \in (-\frac{\pi}{2},0] \) 时，\( \tan x \leq x \)；
    \item 等号条件：当且仅当 \( x=0 \)；
    \item 链式关系：\( \sin x < x < \tan x \)（\( x\in(0,\frac{\pi}{2}) \)）。
\end{enumerate}
