\begin{trapbox}
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.35em]
    \item[\textbf{\textcolor{CTrap}{易错点一（忽略区间条件）}}] 在使用不等式 $\tan x \geq x$ 时，未检查 $x$ 是否属于 $[0, \frac{\pi}{2})$，导致在负区间错误应用。
    \item[\textbf{\textcolor{CTrap}{正确理解（区间限定）：}}] 定理中的不等式仅在指定区间内成立：$\tan x \geq x$ 要求 $x \in [0, \frac{\pi}{2})$；$\tan x \leq x$ 要求 $x \in (-\frac{\pi}{2}, 0]$。
    \item[\textbf{\textcolor{CTrap}{错因分析（条件遗漏）：}}] 记忆不牢，只记住了结论形式，忽略了前提条件；解题时没有养成先验证区间的习惯。
\end{description}

\medskip
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.35em]
    \item[\textbf{\textcolor{CTrap}{易错点二（符号方向混淆）}}] 在负区间 $x<0$ 时，误用 $\tan x \geq x$，而实际上应为 $\tan x \leq x$。
    \item[\textbf{\textcolor{CTrap}{正确理解（符号对称）：}}] 由于 $\tan x$ 是奇函数，在负区间的不等式方向与正区间相反：$x<0$ 时 $\tan x \leq x$，$x>0$ 时 $\tan x \geq x$。
    \item[\textbf{\textcolor{CTrap}{错因分析（奇偶性忽略）：}}] 没有利用函数的奇偶性进行推导，或机械记忆导致符号记反。
\end{description}

\medskip
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.35em]
    \item[\textbf{\textcolor{CTrap}{易错点三（等号滥用）}}] 认为不等式 $\tan x \geq x$ 在 $x \in [0, \frac{\pi}{2})$ 内总是严格成立，或误以为在 $x=\frac{\pi}{2}$ 处等号成立。
    \item[\textbf{\textcolor{CTrap}{正确理解（等号唯一）：}}] 等号仅当 $x=0$ 时成立；在 $x>0$ 时 $\tan x > x$，在 $x<0$ 时 $\tan x < x$。且 $x=\frac{\pi}{2}$ 处 $\tan x$ 无定义。
    \item[\textbf{\textcolor{CTrap}{错因分析（端点混淆）：}}] 对等号成立条件记忆模糊，或混淆了定义域端点与取等点。
\end{description}
\end{trapbox}
