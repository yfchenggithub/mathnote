\begin{explanationbox}
\textbf{\textcolor{CExplanation}{一句话直觉}}
当所有正实数相等时，它们的和与倒数和之积取到最小值 $n^2$；任何不等的情况都会使乘积变大。

\medskip
\textbf{\textcolor{CExplanation}{核心拆解}}
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.45em]
    \item[\textbf{要点一（正实数前提）：}] 所有数必须为正，否则倒数无意义或不等式方向可能改变。
    \item[\textbf{要点二（均值不等式）：}] 通过均值不等式（如 AM-GM）可直接推导，本质是 $n$ 个正数乘积与和的关联。
\end{description}

\medskip
\textbf{\textcolor{CExplanation}{几何本质（算术调和平均）}}
从平均值角度看，不等式等价于算术平均不小于调和平均，即 $\frac{\sum a_i}{n} \geq \frac{n}{\sum \frac{1}{a_i}}$，等号成立时所有数相等。

\medskip
\textbf{\textcolor{CExplanation}{代数意义（通分均值证）}}
代数上，将乘积展开为 $n$ 加上所有交叉项 $\frac{a_i}{a_j}$，利用均值不等式可知每个交叉项不小于 $2$（当 $i=j$ 时为 $1$），从而总和不小于 $n^2$。

\medskip
\textbf{\textcolor{CExplanation}{考点价值（不等式证明）}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{考法一（直接套用）：} 在题目中直接出现和与倒数和的形式时，可直接写出不等式求下界。
    \item \textbf{考法二（变形应用）：} 通过代换或组合，将其他不等式转化为此形式，用于证明或求最值。
\end{itemize}

\medskip
\textbf{\textcolor{CExplanation}{顿悟点}}
\textbf{只要看到正实数的和与倒数和同时出现，就可以尝试应用此不等式，其最小值为 $n^2$。}

\medskip
\textbf{\textcolor{CExplanation}{使用场景}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{场景一（证明不等式）：} 用于证明涉及和与倒数和的不等式，简化推导。
    \item \textbf{场景二（求最小值）：} 当问题要求乘积 $(\sum a_i)(\sum \frac{1}{a_i})$ 的最小值时，直接得 $n^2$。
\end{itemize}
\end{explanationbox}
