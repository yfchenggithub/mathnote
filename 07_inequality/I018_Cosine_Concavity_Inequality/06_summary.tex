\begin{summarybox}
\textbf{\textcolor{CSummary}{一句话核心}}

余弦函数在锐角区间凹，导致两角余弦平均不超过半角余弦；锐角三角形余弦和有上界 $\frac{3}{2}$。

\medskip
\textbf{\textcolor{CSummary}{使用条件}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{条件 1（角度范围）：} $x, y \in (0, \frac{\pi}{2})$。
    \item \textbf{条件 2（三角形内角）：} $A, B, C$ 是锐角三角形的三个内角。
\end{itemize}

\medskip
\textbf{\textcolor{CSummary}{关键提醒}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{易错点（区间与方向）：} 确保角度在锐角区间，并记住不等式方向。
    \item \textbf{检查项（取等条件）：} 等号成立时 $x=y$ 或 $A=B=C=\frac{\pi}{3}$。
\end{itemize}
\end{summarybox}
