\begin{explanationbox}
\textbf{\textcolor{CExplanation}{一句话直觉}}

对数函数图像是“向上凸”的，因此连接曲线上两点的弦总在曲线下方。

\medskip
\textbf{\textcolor{CExplanation}{核心拆解}}
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.45em]
    \item[\textbf{要点一（凹函数定义）：}] 函数 $f(t) = \ln t$ 在 $(0, +\infty)$ 上为凹函数，即其二阶导数为负。
    \item[\textbf{要点二（Jensen不等式）：}] 凹函数满足：函数值的平均不超过平均值的函数值。
\end{description}

\medskip
\textbf{\textcolor{CExplanation}{几何本质（弦在曲线下）}}

对于凹函数，图像上任意两点连成的弦段总位于函数图像之下。因此，弦中点对应的函数值不小于曲线中点的函数值。

\medskip
\textbf{\textcolor{CExplanation}{代数意义（平均不等式）}}

将“对数”视为一种运算，该不等式表明：先取对数再平均，结果不大于先平均再取对数。这类似于算术-几何平均不等式的对数版本。

\medskip
\textbf{\textcolor{CExplanation}{考点价值（不等式证明）}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{考法一（直接应用）：} 用于证明含对数的不等式，或比较对数表达式大小。
    \item \textbf{考法二（函数凸性）：} 结合函数凹凸性，考察 Jensen 不等式的使用。
\end{itemize}

\medskip
\textbf{\textcolor{CExplanation}{顿悟点}}

\textbf{当题目中出现“$\ln\frac{x+y}{2}$”与“$\frac{\ln x+\ln y}{2}$”的比较时，应立即联想到此结论。}

\medskip
\textbf{\textcolor{CExplanation}{使用场景}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{场景一（双变量不等式）：} 涉及两个正实数对数平均的比较问题。
    \item \textbf{场景二（多变量推广）：} 需要处理多个正实数对数表达式时，使用推广形式。
\end{itemize}
\end{explanationbox}
