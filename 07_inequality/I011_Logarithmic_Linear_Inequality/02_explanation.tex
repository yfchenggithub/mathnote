\begin{explanationbox}
\textbf{\textcolor{CExplanation}{一句话直觉}}
对数函数 $y=\ln x$ 在 $x=1$ 处的切线是 $y=x-1$，图像总在切线上方（$x\neq1$）。

\medskip
\textbf{\textcolor{CExplanation}{核心拆解}}
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.45em]
    \item[\textbf{要点一（切线构造）：}] 结论源于导数 $f'(1)=1$ 和切线方程 $y=x-1$。
    \item[\textbf{要点二（函数凹凸性）：}] 因 $f''(x)=-1/x^2<0$，函数图像下凸，故切线在图像下方。
    \item[\textbf{要点三（恒等变形）：}] 通过换元 $t=x+1$ 或两边乘以 $x$ 得到常用变式。
    \item[\textbf{要点四（精度提升）：}] 第三变式用二次函数逼近，比一次切线更紧。
\end{description}

\medskip
\textbf{\textcolor{CExplanation}{几何本质（图像切线）}}
在点 $(1,0)$ 作曲线 $y=\ln x$ 的切线，该切线 $y=x-1$ 总在曲线下方（除切点外）。所有变式都是这一几何事实的坐标伸缩或平移。

\medskip
\textbf{\textcolor{CExplanation}{代数意义（线性逼近）}}
用一次线性函数 $x-1$ 去逼近复杂的对数函数 $\ln x$，并给出一个简单的不等式关系。它是泰勒展开一阶近似的直接体现。

\medskip
\textbf{\textcolor{CExplanation}{考点价值（放缩工具）}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{考法一（证明不等式）：} 用于将对数项放缩为多项式，简化证明。
    \item \textbf{考法二（求最值范围）：} 通过放缩将含对数的最值问题转化为多项式最值。
    \item \textbf{考法三（数列求和放缩）：} 在数列不等式中，常用于裂项求和或比较大小。
\end{itemize}

\medskip
\textbf{\textcolor{CExplanation}{顿悟点}}
\textbf{遇到 $\ln x$ 或 $\ln(1+x)$ 与多项式比较时，优先考虑切线放缩。} 它是沟通超越函数与代数函数的桥梁。

\medskip
\textbf{\textcolor{CExplanation}{使用场景}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{场景一（含参不等式）：} 当不等式含有 $\ln x$ 与 $x$ 的线性组合时。
    \item \textbf{场景二（数列放缩）：} 如证明 $\sum \ln(1+1/n) < 1$ 类问题。
    \item \textbf{场景三（函数最值）：} 求 $f(x)=x\ln x - ax$ 类型函数的最值或范围。
\end{itemize}
\end{explanationbox}
