\section*{1.11 对数切线放缩不等式}
对于任意正实数 \( x \in (0,+\infty) \)：
\begin{enumerate}
    \item 基本形式：\( \ln x \leq x-1 \)；
    \item 等号成立条件：当且仅当 \( x=1 \)；
    \item 常用变式：
    \[
    \ln(x+1) \leq x\;(x>-1),\quad x\ln x \geq x-1\;(x>0),\quad \ln x \leq \frac{x^2-1}{2x}.
    \]
\end{enumerate}
