\begin{explanationbox}
\textbf{\textcolor{CExplanation}{一句话直觉}}
正切函数在 $(0, \pi/2)$ 区间内是凸函数，因此两个角的正切值的算术平均不会小于这两角算术平均的正切值；推广到锐角三角形，三个内角正切之和有一个确定的下界 $3\sqrt{3}$。

\medskip
\textbf{\textcolor{CExplanation}{核心拆解}}
\begin{description}[style=nextline, leftmargin=2.8em, labelsep=0.5em, itemsep=0.45em]
    \item[\textbf{要点一（凸性定义）：}] 函数 $f(x)$ 在区间 $I$ 上称为凸函数，如果对于任意 $x_1, x_2 \in I$ 和 $t \in [0,1]$，有 $f(tx_1 + (1-t)x_2) \leq t f(x_1) + (1-t) f(x_2)$。正切函数在 $(0, \pi/2)$ 满足此性质。
    \item[\textbf{要点二（Jensen不等式）：}] 对于凸函数 $f$，有 $f\left(\frac{x+y}{2}\right) \leq \frac{f(x)+f(y)}{2}$，这正是主结论的表达式。
\end{description}

\medskip
\textbf{\textcolor{CExplanation}{几何本质（正切曲线凸性）}}
在 $(0, \pi/2)$ 区间，正切曲线的图像是向下凸的（即二阶导数为正），因此连接曲线上两点的线段位于曲线上方，导致算术平均的正切值不大于正切值的算术平均。

\medskip
\textbf{\textcolor{CExplanation}{代数意义（差法证明）}}
可通过考虑差函数 $D(x,y) = \frac{\tan x + \tan y}{2} - \tan\left(\frac{x+y}{2}\right)$，利用三角恒等式化简，证明其非负，从而建立不等式。

\medskip
\textbf{\textcolor{CExplanation}{考点价值（不等式最值）}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{考法一（直接应用）：} 在比较两个角的正切值大小或证明相关不等式时，直接使用该结论。
    \item \textbf{考法二（三角形最值）：} 在锐角三角形中，求正切和的最小值或证明其不小于 $3\sqrt{3}$，常用于解三角形与不等式综合题。
\end{itemize}

\medskip
\textbf{\textcolor{CExplanation}{顿悟点}}
\textbf{当遇到正切函数在锐角范围内的不等式时，可优先考虑其凸性，利用 Jensen 不等式或差法快速证明。}

\medskip
\textbf{\textcolor{CExplanation}{使用场景}}
\begin{itemize}[leftmargin=*, itemsep=0.5em, topsep=0.4em]
    \item \textbf{场景一（两角比较）：} 已知两个锐角，比较 $\tan x + \tan y$ 与 $2\tan((x+y)/2)$ 的大小。
    \item \textbf{场景二（三角形论证）：} 在锐角三角形中，证明 $\tan A + \tan B + \tan C \geq 3\sqrt{3}$ 或类似结论。
\end{itemize}
\end{explanationbox}
