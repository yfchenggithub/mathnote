\begin{explanationbox}
德摩根公式的本质是"否定"与"且"、"或"之间的逻辑等价关系：
\begin{itemize}
    \item \textbf{逻辑视角}：$x \notin (A \cap B)$ 等价于 $x \notin A$ 或 $x \notin B$，即 $x \in C_U A \cup C_U B$；
    \item \textbf{对称性}：两个公式互为对偶，只需将一个中的 $\cap$ 与 $\cup$ 互换，并交换左右两边即得另一个；
    \item \textbf{推广}：公式可推广到任意多个集合的并和交：
    \[
    C_U\left(\bigcap_{i=1}^n A_i\right) = \bigcup_{i=1}^n C_U A_i,\quad
    C_U\left(\bigcup_{i=1}^n A_i\right) = \bigcap_{i=1}^n C_U A_i.
    \]
\end{itemize}
\end{explanationbox}
