\begin{trapbox}
\begin{itemize}
    \item \textbf{方向混淆}：$A\subseteq B$ 等价于 $C_U B \subseteq C_U A$，注意补集的包含关系与原集合相反。容易误记为 $C_U A \subseteq C_U B$。
    \item \textbf{与真子集混淆}：$A\cap B = A$ 只说明 $A\subseteq B$，不保证 $A\neq B$。同理，$A\cup B = B$ 也不排除 $A=B$ 的情况。
    \item \textbf{忽略全集}：补集的定义依赖于全集 $U$。在应用 $C_U B \subseteq C_U A$ 时，必须明确 $U$ 是什么，否则结论可能不成立。
    \item \textbf{逻辑跳跃}：在解题中，直接由 $A\cap B = A$ 推出 $A\subseteq B$ 是常用的，但需要理解其推导依据，不可死记硬背。
    \item \textbf{符号滥用}：有些资料用 $A\subset B$ 表示真子集，但本文中 $A\subseteq B$ 表示子集（可能相等）。注意区分不同教材的符号约定。
\end{itemize}
\end{trapbox}
