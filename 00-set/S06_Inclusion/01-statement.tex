\begin{statementbox}
集合的包含关系与交、并、补运算之间存在如下等价链条（设 $A,B$ 为全集 $U$ 的子集）：
\[
A\cap B = A \quad\Leftrightarrow\quad A\cup B = B \quad\Leftrightarrow\quad A\subseteq B \quad\Leftrightarrow\quad C_U B \subseteq C_U A.
\]
即下列四个命题相互等价：
\begin{enumerate}
    \item $A$ 与 $B$ 的交集等于 $A$；
    \item $A$ 与 $B$ 的并集等于 $B$；
    \item $A$ 是 $B$ 的子集；
    \item $B$ 的补集是 $A$ 的补集的子集。
\end{enumerate}
\end{statementbox}
