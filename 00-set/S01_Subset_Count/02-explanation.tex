\begin{explanationbox}
集合子集个数的代数本质是"乘法原理"与"二项式系数之和"：
\begin{itemize}
    \item \textbf{分步计数}：对集合中的每个元素，它在子集中有两种状态——属于或不属于。由乘法原理，所有可能的状态组合数为 $2^n$，这就是子集总数。
    \item \textbf{与二项式系数的关系}：子集个数也可按元素个数分类：$k$ 元子集个数为 $\mathrm{C}_n^k$，故总数为 $\sum_{k=0}^n \mathrm{C}_n^k = 2^n$。
    \item \textbf{真子集与非空子集}：真子集不包括原集合本身，因此个数为 $2^n-1$；非空子集不包括空集，因此个数也为 $2^n-1$；同时满足"非空"和"真"的子集个数为 $2^n-2$。
\end{itemize}
\end{explanationbox}
