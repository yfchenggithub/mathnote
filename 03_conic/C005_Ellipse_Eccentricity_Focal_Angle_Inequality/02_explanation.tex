\begin{explanationbox}
该结论是连接椭圆"扁平程度"与"视角"的桥梁：
\begin{itemize}
    \item \textbf{几何直观}：离心率 $e$ 越大，椭圆越扁，焦点越靠近顶点，动点 $P$ 对焦点的"视野"就越开阔，最大张角也就越大。
    \item \textbf{结构转化}：通过 $\sin \frac{\theta}{2}$ 的形式，将复杂的三角函数范围问题直接简化为对 $e$ 的代数约束。
    \item \textbf{适用场景}：
    \begin{itemize}
        \item 已知张角的范围，求离心率 $e$ 的取值范围。
        \item 证明椭圆上是否存在满足特定角度条件的点。
        \item 处理涉及三角形 $F_1PF_2$ 的外接圆或内切圆半径的最值问题。
    \end{itemize}
\end{itemize}
\end{explanationbox}
