\begin{proofbox}
\textbf{利用正切半角公式推导：}
\begin{enumerate}
    \item 在 $\triangle F_1PF_2$ 中，设 $P(x_0, y_0)$。根据椭圆定义及余弦定理可推导，但最简便方法是利用直角三角形投影：
    \item $\tan \angle PF_2O = \frac{y_0}{c-x_0}$，$\tan \angle PF_1O = \frac{y_0}{c+x_0}$（假设 $P$ 在第一象限）。
    \item 利用恒等式 $\tan \theta = \tan(\angle PF_1F_2 + \angle PF_2F_1)$：
    $\tan \theta = \frac{\frac{y_0}{c+x_0} + \frac{y_0}{c-x_0}}{1 - \frac{y_0^2}{c^2-x_0^2}} = \frac{2cy_0}{c^2 - x_0^2 - y_0^2}$。
    \item 代入 $x_0^2 = a^2(1 - \frac{y_0^2}{b^2})$，整理得 $\tan \theta = \frac{2cy_0}{b^2 - a^2 + y_0^2(1 - \frac{a^2}{b^2})} = \frac{2cy_0}{y_0^2 \cdot \frac{c^2}{b^2} - c^2}$。
    \item 进一步分析可知，$\theta$ 随 $y_0$ 的增大而增大，故 $y_0 = b$ 时角度最大。
\end{enumerate}
\end{proofbox}
