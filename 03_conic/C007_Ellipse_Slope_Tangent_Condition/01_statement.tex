\begin{statementbox}
对于椭圆 $C: \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ ($a>b>0$) 与直线 $l: y = kx + m$. 直线与椭圆相切的充分必要条件（即切线判定公式）为：
\begin{enumerate}
    \item \textbf{核心判定式}：$m^2 = a^2k^2 + b^2$;
    \item \textbf{位置关系推论}：
    \begin{itemize}
        \item 若 $m^2 < a^2k^2 + b^2$, 则直线与椭圆相交;
        \item 若 $m^2 > a^2k^2 + b^2$, 则直线与椭圆相离.
    \end{itemize}
    \item \textbf{切线方程}：已知斜率为 $k$ 的椭圆切线方程可直接设为 $y = kx \pm \sqrt{a^2k^2 + b^2}$.
\end{enumerate}
\end{statementbox}
