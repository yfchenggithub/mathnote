\begin{examplebox}
\textbf{例题 1}: 求斜率为 $1$ 且与椭圆 $\frac{x^2}{4} + y^2 = 1$ 相切的直线方程.
\textit{解析}: 
1. 由题意知 $a^2=4, b^2=1, k=1$.
2. 由判定公式 $m^2 = a^2k^2 + b^2 = 4(1)^2 + 1 = 5$.
3. 故 $m = \pm \sqrt{5}$.
4. 所求切线方程为 $y = x + \sqrt{5}$ 或 $y = x - \sqrt{5}$.

\textbf{例题 2}: 过点 $M(3, 0)$ 引椭圆 $x^2 + 2y^2 = 2$ 的切线, 求切线斜率.
\textit{解析}:
1. 椭圆标准方程为 $\frac{x^2}{2} + y^2 = 1$, 则 $a^2=2, b^2=1$.
2. 设切线方程为 $y = k(x-3)$, 即 $y = kx - 3k$, 则 $m = -3k$.
3. 代入公式 $(-3k)^2 = 2k^2 + 1 \implies 9k^2 = 2k^2 + 1$.
4. $7k^2 = 1 \implies k = \pm \frac{\sqrt{7}}{7}$.
\end{examplebox}
