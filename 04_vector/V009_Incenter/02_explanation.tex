\begin{explanationbox}
内心结论的代数特征是"边长加权平均"：
\begin{itemize}
    \item \textbf{权值分配}：与重心（系数均为1）不同，内心的向量系数恰好等于对应顶点的对边边长。
    \item \textbf{几何直观}：由于角平分线定理规定了对边被平分线段之比等于邻边之比，这导致了向量合成时必须引入边长作为权重进行平衡。
    \item \textbf{奔驰定理联系}：在奔驰定理中，当面积比 $S_A : S_B : S_C = a : b : c$ 时，该点即为内心。
\end{itemize}
\end{explanationbox}
