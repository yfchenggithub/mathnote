\begin{trapbox}
\begin{itemize}
    \item \textbf{系数对应错误}：内心公式中 $\vec{IA}$ 前面的系数必须是 $A$ 的对边 $a$（即 $BC$ 的长度）。学生常错记为 $b$ 或 $c$。
    \item \textbf{单位向量陷阱}：角平分线方向常表示为 $\frac{\vec{AB}}{|\vec{AB}|} + \frac{\vec{AC}}{|\vec{AC}|}$，这本质上是利用单位向量构造菱形，其结果与内心结论相通，需注意区分。
    \item \textbf{未化简等式}：若给出的等式右边不是 $\vec{0}$，务必先进行移项。
\end{itemize}
\end{trapbox}
