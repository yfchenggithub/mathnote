\begin{examplebox}
\textbf{典型例题}：
在 $\triangle ABC$ 中，$M$ 是 $BC$ 的中点，点 $P$ 满足 $\vec{AP} = \lambda \vec{AB} + \mu \vec{AC}$。若点 $P$ 落在中线 $AM$ 上，求 $\lambda$ 与 $\mu$ 的关系。

\textit{解析}：
1. 找到 $k=1$ 的基准线：直线 $BC$。在基准线上 $\lambda + \mu = 1$。
2. 观察等和线：点 $P$ 落在过 $A$ 的直线 $AM$ 上。但这并不符合平行关系。
3. \textbf{注意}：等和线定理要求等和线\textbf{平行}于基准线。
4. \textbf{修正思路}：若 $P$ 落在过 $M$ 且平行于 $AB$ 的直线上，则可直接使用。
5. 本题通常考查三点共线：若 $P, B, C$ 共线，则 $\lambda + \mu = 1$。若 $\vec{AP} = k\vec{AM} = k(\frac{1}{2}\vec{AB} + \frac{1}{2}\vec{AC})$，则 $\lambda = \mu = \frac{k}{2}$，即 $\lambda = \mu$。
\end{examplebox}
