\begin{trapbox}
\begin{itemize}
    \item \textbf{起点必须统一}：公式中的三个向量 $\vec{AP}, \vec{AM}, \vec{AN}$ 必须拥有相同的起点。
    \item \textbf{系数和为1的条件}：只有当三个向量的终点共线时，系数和才为1。如果起点 $A$ 也在该直线上，结论依然成立，但向量将退化为共线向量。
    \item \textbf{注意 $1-\lambda$}：在填空题中，常有 $\vec{AP} = x\vec{AM} + y\vec{AN}$ 的形式，务必核对 $x+y=1$。
\end{itemize}
\end{trapbox}
