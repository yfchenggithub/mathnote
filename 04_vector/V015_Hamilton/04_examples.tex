\begin{examplebox}
\textbf{例题}：在 $\triangle ABC$ 中，外接圆半径 $R=1$，三边长 $a, b, c$ 满足 $a^2+b^2+c^2 = 8$，求外心 $O$ 到垂心 $H$ 的距离。

\textbf{解析}：
1. 直接使用模长推论公式：
   $|\vec{OH}|^2 = 9R^2 - (a^2 + b^2 + c^2)$
2. 代入已知数据：
   $|\vec{OH}|^2 = 9(1)^2 - 8 = 1$
3. 解得：$|\vec{OH}| = 1$。
4. \textbf{拓展}：此时可以进一步求得重心到外心的距离 $OG = \frac{1}{3}OH = \frac{1}{3}$。
\end{examplebox}
