\begin{statementbox}
\textbf{结论：九点圆（费尔巴哈圆）定理}
对于任意 $\triangle ABC$，以下九个点共圆：
\begin{enumerate}
    \item \textbf{三边中点}：$D, E, F$；
    \item \textbf{三条高的垂足}：$H_1, H_2, H_3$；
    \item \textbf{垂心与顶点连线的中点}（欧拉点）：$X, Y, Z$。
\end{enumerate}
\textbf{核心性质}：
\begin{itemize}
    \item \textbf{半径关系}：九点圆半径 $R_9 = \frac{1}{2}R$（$R$ 为外接圆半径）；
    \item \textbf{圆心位置}：九点圆心 $E$ 是外心 $O$ 与垂心 $H$ 的中点；
    \item \textbf{费尔巴哈定理}：九点圆与内切圆内切，与三个旁切圆外切。
\end{itemize}
\end{statementbox}

\begin{statementbox}
\textbf{结论：九点圆心位置特性}
设 $\triangle ABC$ 的外心为 $O$，垂心为 $H$，九点圆圆心为 $E$，则：
\begin{enumerate}
    \item \textbf{中点性质}：$\vec{OE} = \vec{EH} = \frac{1}{2}\vec{OH}$；
    \item \textbf{坐标表示}：$E = \frac{O + H}{2}$；
    \item \textbf{向量关系}：对于平面内任意一点 $P$，$\vec{PE} = \frac{1}{2}(\vec{PO} + \vec{PH})$；
    \item \textbf{半径对应}：九点圆的圆心到九点的距离恒为 $\frac{1}{2}R$。
\end{enumerate}
\end{statementbox}
