重心结论的代数本质是"算术平均值"：
\begin{itemize}
    \item \textbf{平衡点}：$\vec{GA} + \vec{GB} + \vec{GC} = \vec{0}$ 说明重心是三角形三个顶点向量系统的平衡中心。
    \item \textbf{奔驰定理特例}：当奔驰定理系数 $x:y:z = 1:1:1$ 时，该点即为重心。
    \item \textbf{线性坐标}：在以 $O$ 为原点的坐标系中，重心的坐标即为三个顶点坐标的平均值，这为数形结合提供了极大的便利。
\end{itemize}
\end{explanationbox}
