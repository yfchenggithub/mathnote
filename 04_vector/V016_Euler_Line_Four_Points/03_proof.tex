\begin{proofbox}
\textbf{欧拉线共线与比例的向量证明}：
\begin{enumerate}
    \item \textbf{建立基准}：由哈密顿公式知，以角外心 $O$ 为起点，有 $\vec{OH} = \vec{OA} + \vec{OB} + \vec{OC}$。
    \item \textbf{引入重心}：根据重心性质，$\vec{OG} = \frac{1}{3}(\vec{OA} + \vec{OB} + \vec{OC})$。
    \item \textbf{证明 $O, G, H$ 共线}：显然 $\vec{OH} = 3\vec{OG}$。因为向量共线且有公共点 $O$，故 $O, G, H$ 三点共线，且 $H$ 在 $OG$ 延长线上，$GH = 2OG$。
    \item \textbf{引入九点圆心}：九点圆心 $E$ 是 $OH$ 的中点，即 $\vec{OE} = \frac{1}{2}\vec{OH}$。
    \item \textbf{计算 $GE$}：$\vec{GE} = \vec{OE} - \vec{OG} = \frac{1}{2}\vec{OH} - \frac{1}{3}\vec{OH} = \frac{1}{6}\vec{OH}$。
    \item \textbf{结论}：四点均可由 $\vec{OH}$ 线性表示，故四点共线。
\end{enumerate}
\end{proofbox}
