\begin{proofbox}
\textbf{等和线定理简要证明}：
\begin{enumerate}
    \item 设直线 $AB$ 的方程为 $l_0$。若点 $P$ 在直线 $AB$ 上，根据三点共线定理有 $\lambda + \mu = 1$。
    \item 设直线 $l$ 平行于 $AB$，则可设其方程为 $\vec{OP} = k \vec{OQ}$，其中 $Q$ 是直线 $AB$ 上的一点。
    \item 令 $\vec{OQ} = \lambda_0 \vec{OA} + \mu_0 \vec{OB}$，且 $\lambda_0 + \mu_0 = 1$。
    \item 则 $\vec{OP} = k(\lambda_0 \vec{OA} + \mu_0 \vec{OB}) = (k\lambda_0)\vec{OA} + (k\mu_0)\vec{OB}$。
    \item 记 $\lambda = k\lambda_0, \mu = k\mu_0$，则有 $\lambda + \mu = k(\lambda_0 + \mu_0) = k \cdot 1 = k$。
    \item 因 $l // AB$，由 $k$ 的比例性质可知，该常数 $k$ 即为直线 $l$ 与直线 $AB$ 到 $O$ 点距离的比值。
\end{enumerate}
\end{proofbox}
