\begin{statementbox}
\textbf{结论：三角形的旁心结论}
设 $O$ 为 $\triangle ABC$ 的 $\angle A$ 的旁心（即内角 $\angle A$ 平分线与外角 $\angle B, \angle C$ 平分线的交点），角 $A, B, C$ 对应的边长分别为 $a, b, c$，则：
\begin{enumerate}
    \item \textbf{基准等式}：$-a\vec{OA} + b\vec{OB} + c\vec{OC} = \vec{0}$；
    \item \textbf{任意点表示}：对于平面内任意一点 $M$，有 $\vec{MO} = \frac{-a\vec{MA} + b\vec{MB} + c\vec{MC}}{-a+b+c}$；
    \item \textbf{单位向量表示}：向量 $\vec{v} = \lambda \left( \frac{\vec{AB}}{|\vec{AB}|} + \frac{\vec{AC}}{|\vec{AC}|} \right)$ 指向旁心 $O$ 所在直线。
\end{enumerate}
\end{statementbox}
