\begin{itemize}
    \item \textbf{系数正负}：很多同学会将 $-a\vec{OA}$ 记错位置。记住：\textbf{哪个角的旁心，哪个顶点系数就变负}。
    \item \textbf{分母求和}：在任意点表示公式中，分母是系数的代数和 $-a+b+c$，由于三角形两边之和大于第三边，这个分母永远不为 0。
    \item \textbf{单位向量混淆}：$\frac{\vec{AB}}{c} + \frac{\vec{AC}}{b}$ 指向的是内心和旁心共有的这条角平分线射线。
\end{itemize}
\end{trapbox}
