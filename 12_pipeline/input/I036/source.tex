\section*{1.36 对数函数分式下界}
对于任意实数 \( x > -1 \)：
\begin{enumerate}
    \item 核心形式：\( \ln(1+x) \geq \frac{x}{1+x} \)；
    \item 链式关系：\( \frac{x}{1+x} \leq \ln(1+x) \leq x \)；
    \item 等号条件：当且仅当 \( x=0 \)；
    \item 变式形式：令 \( 1+x=t>0 \)，则 \( \ln t \geq 1-\frac{1}{t} \)。
\end{enumerate}
