\section*{1.35 对数函数二阶放缩}
对于 \( x \geq 0 \)：
\begin{enumerate}
    \item 核心形式：\( \ln(1+x) \leq x - \frac{x^2}{2} + \frac{x^3}{3} \)（三阶项视精度）；
    \item 常用二阶形式：\( \ln(1+x) \leq x - \frac{x^2}{2} \) 仅在局部成立；更稳健的为 \( \frac{2x}{x+2} \leq \ln(1+x) \leq \frac{x(x+2)}{2(x+1)} \)；
    \item 标准二阶下界：\( \ln(1+x) \geq x - \frac{x^2}{2(1+x)} \)；
    \item 等号条件：当且仅当 \( x=0 \)。
\end{enumerate}
