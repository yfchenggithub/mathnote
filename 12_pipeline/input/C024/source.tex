\section*{椭圆周长的高精度近似公式}

\[
\frac{x^2}{a^2}+\frac{y^2}{b^2}=1
\qquad (a\ge b>0)
\]

的周长记为 \(C\)。令

\[
h=\left(\frac{a-b}{a+b}\right)^2,
\]

则椭圆周长可近似表示为

\[
\boxed{
C\approx
\pi(a+b)
\left(
1+\frac{3h}{10+\sqrt{4-3h}}
\right)
}
\]

即

\[
\boxed{
C\approx
\pi(a+b)
\left[
1+
\frac{
3\left(\frac{a-b}{a+b}\right)^2
}{
10+\sqrt{
4-3\left(\frac{a-b}{a+b}\right)^2
}
}
\right]
}
\]

这是 Ramanujan 椭圆周长近似公式，精度很高，尤其适合在高中数学中作为椭圆周长的常用二级结论使用。

当 \(a=b=r\) 时，

\[
h=0,
\]

于是

\[
C\approx \pi(2r)=2\pi r,
\]

与圆的周长公式一致。
