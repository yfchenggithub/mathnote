\subsection{柯西不等式}

\Conclusion

设 \(a_i, b_i \in \mathbb{R}\)，柯西不等式在高考中常见的表现形式如下：

\begin{enumerate}
    \item \textbf{二维形式}：
    \[ (a^2 + b^2)(c^2 + d^2) \geq (ac + bd)^2 \]

    \item \textbf{三维形式}：
    \[ (a_1^2 + a_2^2 + a_3^2)(b_1^2 + b_2^2 + b_3^2) \geq (a_1b_1 + a_2b_2 + a_3b_3)^2 \]

    \item \textbf{一般形式}：
    \[ \left(\sum_{i=1}^n a_i^2\right)\left(\sum_{i=1}^n b_i^2\right) \geq \left(\sum_{i=1}^n a_i b_i\right)^2 \]

    \item \textbf{向量形式}：
    \[ |\vec{a}|^2 \cdot |\vec{b}|^2 \geq (\vec{a} \cdot \vec{b})^2 \]

    \item \textbf{等号成立条件}：当且仅当序列 \((a_i)\) 与 \((b_i)\) 成比例（即 \(a_i b_j = a_j b_i\)）时成立。
\end{enumerate}
