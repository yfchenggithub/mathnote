\subsection{糖水不等式及其应用}

\Conclusion

对于正实数 \(a, b, m \in \mathbb{R}^+\), 且 \(a < b\), 有以下分式不等式结论（俗称“糖水不等式”）：

\begin{enumerate}
    \item \textbf{性质 A}（加糖变甜）：
    \[ \frac{a}{b} < \frac{a+m}{b+m} \]

    \item \textbf{性质 B}（减糖变淡）：
    \[ \frac{a}{b} > \frac{a-m}{b-m} \quad (\text{其中 } b > m) \]

    \item \textbf{推广形式}：若 \(b > a > 0\), 则
    \[ \frac{b}{a} > \frac{b+m}{a+m} \]
\end{enumerate}
