\section*{1.23 \(x\ln x\) 凸性放缩}
对于任意正实数 \( x,y>0 \)：
\begin{enumerate}
    \item 核心形式：\( \frac{x\ln x+y\ln y}{2} \geq \frac{x+y}{2}\ln\!\left(\frac{x+y}{2}\right) \)；
    \item 等号成立条件：当且仅当 \( x=y \)；
    \item 推广形式：\( \frac{\sum x_i\ln x_i}{n} \geq \left(\frac{\sum x_i}{n}\right)\ln\!\left(\frac{\sum x_i}{n}\right) \)。
\end{enumerate}
