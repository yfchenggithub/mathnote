\section*{1.29 正弦函数基础放缩}
对于任意实数 \( x \geq 0 \)：
\begin{enumerate}
    \item 核心形式：\( \sin x \leq x \)；
    \item 对称形式：当 \( x \leq 0 \) 时，\( \sin x \geq x \)；综上 \( |\sin x| \leq |x| \)；
    \item 等号条件：当且仅当 \( x=0 \)。
\end{enumerate}
