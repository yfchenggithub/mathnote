\section*{1.1 解连不等式的转化形式}

对于连不等式 \( N < f(x) < M \)（其中 \( M > N \)），其解集可以通过以下多种等价形式进行转化，以便于处理不同类型的题目：

\begin{enumerate}
    \item \textbf{乘积形式}：\( N < f(x) < M \iff [f(x) - M][f(x) - N] < 0 \)；
    
    \item \textbf{绝对值形式}：\( N < f(x) < M \iff \left| f(x) - \frac{M+N}{2} \right| < \frac{M-N}{2} \)；
    
    \item \textbf{分式形式}：\( N < f(x) < M \iff \frac{f(x)-N}{M-f(x)} > 0 \)；
    
    \item \textbf{倒数形式}：\( N < f(x) < M \iff \frac{1}{f(x)-N} > \frac{1}{M-N} \)。
\end{enumerate}
