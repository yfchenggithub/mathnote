\begin{explanationbox}
包含关系的等价链条反映了子集概念在不同运算下的刻画：
\begin{itemize}
    \item \textbf{$A\cap B = A$ 的意义}：$A$ 的所有元素都在 $B$ 中，即 $A\subseteq B$。因为 $A\cap B\subseteq A$ 恒成立，等式 $A\cap B = A$ 等价于 $A\subseteq A\cap B$，即 $A$ 的每个元素也属于 $B$。
    \item \textbf{$A\cup B = B$ 的意义}：$B$ 包含了 $A$ 的所有元素，即 $A\subseteq B$。因为 $B\subseteq A\cup B$ 恒成立，等式 $A\cup B = B$ 等价于 $A\cup B\subseteq B$，即 $A$ 的元素全在 $B$ 中。
    \item \textbf{$C_U B \subseteq C_U A$ 的意义}：补集的大小关系与原集合的包含关系相反（对偶性）。$A\subseteq B$ 意味着不在 $B$ 中的元素一定不在 $A$ 中，即 $C_U B \subseteq C_U A$。
    \item 这四个命题环环相扣，任意一个成立则其他三个也成立，体现了子集概念的内在一致性。
\end{itemize}
\end{explanationbox}
