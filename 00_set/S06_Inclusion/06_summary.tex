\begin{summarybox}
1. \textbf{核心等价链条}：
   \[
   A\cap B = A \;\Leftrightarrow\; A\cup B = B \;\Leftrightarrow\; A\subseteq B \;\Leftrightarrow\; C_U B \subseteq C_U A.
   \]
2. \textbf{记忆技巧}：交等于小集，并等于大集，补集方向反转。
3. \textbf{应用场景}：常用于集合关系推导、化简集合表达式、证明集合恒等式、判断包含关系等。
4. \textbf{注意事项}：始终明确全集 $U$；区分子集与真子集；理解每个等价的直观含义，避免机械记忆。
\end{summarybox}
