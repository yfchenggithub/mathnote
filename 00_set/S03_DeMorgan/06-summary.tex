\begin{summarybox}
1. \textbf{核心公式}：
   \begin{itemize}
       \item $C_U(A \cap B) = C_U A \cup C_U B$；
       \item $C_U(A \cup B) = C_U A \cap C_U B$。
   \end{itemize}
2. \textbf{记忆口诀}："交之补等于补之并，并之补等于补之交"。
3. \textbf{应用场景}：常用于集合化简、证明集合恒等式、概率论中的事件运算、逻辑推理等。
4. \textbf{注意事项}：公式对任意多个集合也成立，使用时注意运算优先级，补集符号可看作一个整体。
\end{summarybox}
