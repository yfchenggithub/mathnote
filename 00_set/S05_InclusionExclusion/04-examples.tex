\begin{examplebox}
\textbf{例题1：} 某班级有 30 名学生，其中 18 人喜欢数学，15 人喜欢物理，10 人既喜欢数学又喜欢物理。问有多少人至少喜欢一门学科？有多少人两门都不喜欢？

\textit{解析：} 设 $A$ = 喜欢数学的学生，$B$ = 喜欢物理的学生。则 $|A|=18$，$|B|=15$，$|A \cap B|=10$。
\begin{itemize}
    \item 至少喜欢一门的人数：$|A \cup B| = |A| + |B| - |A \cap B| = 18+15-10 = 23$。
    \item 两门都不喜欢的人数：$|U| - |A \cup B| = 30 - 23 = 7$（$U$ 为全班）。
\end{itemize}
\end{examplebox}

\begin{examplebox}
\textbf{例题2：} 从 1 到 100 的整数中，有多少个数能被 2 或 3 或 5 整除？

\textit{解析：} 设 $A$ 为能被 2 整除的数，$|A| = \lfloor 100/2 \rfloor = 50$；$B$ 为能被 3 整除的数，$|B| = \lfloor 100/3 \rfloor = 33$；$C$ 为能被 5 整除的数，$|C| = 20$。
两两交集：$|A \cap B| = \lfloor 100/6 \rfloor = 16$；$|A \cap C| = \lfloor 100/10 \rfloor = 10$；$|B \cap C| = \lfloor 100/15 \rfloor = 6$。
三个交集：$|A \cap B \cap C| = \lfloor 100/30 \rfloor = 3$。
由容斥原理：
\[
|A \cup B \cup C| = 50+33+20 - (16+10+6) + 3 = 103 - 32 + 3 = 74.
\]
故有 74 个数能被 2 或 3 或 5 整除。
\end{examplebox}
