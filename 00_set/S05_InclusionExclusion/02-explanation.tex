\begin{explanationbox}
容斥原理的核心思想是"逐步修正重复计数"：
\begin{itemize}
    \item \textbf{直观理解}：当计算并集的元素个数时，直接相加会多算交集部分，因此减去一次交集；对于三个集合，两两交集被多减了，需要加回三个集合的交集。
    \item \textbf{符号规律}：每一项的符号由集合个数决定，奇数个集合的交集取正，偶数个取负（在一般公式中体现为 $(-1)^{k-1}$）。
    \item \textbf{推广}：容斥原理可以推广到任意有限个集合，也适用于概率论中事件并的概率计算。
\end{itemize}
\end{explanationbox}
