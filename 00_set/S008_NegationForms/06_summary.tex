\begin{summarybox}
1. \textbf{核心规律}：
   \begin{itemize}
       \item 全称量词与存在量词互换，并否定谓词。
       \item 且与或互换，并分别否定每个分支（德摩根定律）。
       \item 比较关系：$>$ 的否定为 $\le$；$<$ 的否定为 $\ge$；$\ge$ 的否定为 $<$；$\le$ 的否定为 $>$。
       \item 数量词："至少 $n$" 的否定为 "至多 $n-1$"；"至多 $n$" 的否定为 "至少 $n+1$"。
   \end{itemize}
2. \textbf{记忆口诀}：量词互换，且或互变，比较取补，数量加减一。
3. \textbf{应用场景}：命题的否定、反证法、构造反例、逻辑推理。
4. \textbf{注意事项}：注意边界条件（等号的归属），分清联结词，避免双重否定处理错误。
\end{summarybox}
