\begin{trapbox}
\begin{itemize}
    \item \textbf{空集易遗漏}：空集是任何集合的子集，也是任何非空集合的真子集。在计算子集个数时，不要忘记空集的存在。
    \item \textbf{"真子集"与"非空子集"混淆}：真子集不含原集合本身，非空子集不含空集，两者个数可能相等（当原集合非空时），但含义完全不同。注意审题，明确题目要求的是"真子集"还是"非空子集"。
    \item \textbf{当 $n=0$ 时}：空集的子集个数为 $2^0=1$（即空集本身），此时不存在非空子集，也不存在真子集（因为空集没有真子集）。上述公式中 $2^n-1$ 在 $n=0$ 时变为 $0$，符合实际，但 $2^n-2$ 会得到负数，因此对空集不能直接使用非空真子集公式，需单独讨论。
    \item \textbf{指数记错}：子集个数公式为 $2^n$，不要误记为 $2n$ 或 $n^2$。
\end{itemize}
\end{trapbox}
