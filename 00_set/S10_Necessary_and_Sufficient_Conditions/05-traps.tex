\begin{itemize}
    \item \textbf{主客体混淆}：务必分清谁是条件、谁是结论。题目问"$p$ 是 $q$ 的什么条件"，则研究的是 $p \rightarrow q$ 和 $q \rightarrow p$。
    \item \textbf{忽略"不充分"或"不必要"}：在判定时，很多学生只证明了一个方向就下结论。必须通过"双向检测"来确定最终关系。
    \item \textbf{否定命题判定的混淆}：处理"不、非、无"等否定词命题时，建议先利用逆否命题进行转化，再进行判定。
    \item \textbf{特例漏选}：在涉及不等式或方程根的分布时（如例 2），要注意端点值（$f(k)=0$）等特殊情况对充分性的影响。
\end{itemize}
